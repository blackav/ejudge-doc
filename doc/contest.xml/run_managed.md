| **Имя атрибута**:          | **`run_managed`**                           |     |
| **Содержится в:**          | [`contest`](contest.xml:contest.md) |     |
| **Тип значения:**          | *boolean*                                   |     |
| **Может отсутствовать:**   | *да*                                        |     |
| **Значение по умолчанию:** | false                                       |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Various contest's flags"*,
поле *"User ej-super-run for testing?"*.

**Описание.**Если значение этого атрибута установлено в *true*,
программа проверки решений [`run`](run.md) ставится под
управление программы [`super-serve`](super-serve.md), которая
следит одновременно за несколькими турнирами и запускает программу run,
как только поступает запрос на проверку решения в соответствующем
турнире. Если значение атрибута установлено в *false*, данный турнир
игнорируется программой `super-serve`.
