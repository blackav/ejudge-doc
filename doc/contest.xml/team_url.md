**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Participation settings"*,
поле *"URL for the \`team' CGI program"*.

**Описание.** Данный элемент позволяет задать URL для CGI-программы
[`team`](../team.md), которая позволяет пользователю
непосредственно участвовать в турнире. Если данный элемент задан, то
ссылка на указанный URL будет выведена на личной странице участника
после успешной регистрации участника в данном турнире, то есть когда
статус регистрации участника установлен в `OK`. Если элемент не задан,
ссылка не демонстрируется.

**Пример.**

`<team_url>http://www.supercontest.ru/cgi-bin/team</team_url>`
