|                            |                                             |     |
|----------------------------|---------------------------------------------|-----|
| **Имя атрибута**:          | **`enable_reminders`**                      |     |
| **Содержится в:**          | [`contest`](contest.xml:contest.md) |     |
| **Тип значения:**          | *boolean*                                   |     |
| **Может отсутствовать:**   | *да*                                        |     |
| **Значение по умолчанию:** | false                                       |     |

**Описание.** Если этот атрибут установлен в *true* или *yes*, каждые
сутки, примерно в 10:00 рассылается уведомление о неотвеченных вопросах
пользователей, непроверенных посылках и т. п. До версии 3.9.0 такое
уведомление рассылалось по умолчанию. Начиная с 3.9.0 по умолчанию
рассылка не производится, и чтобы ее разрешить требуется установить этот
атрибут.

Поддерживается с версии [3.9.0](изменения_в_версии_3.9.0.md).
