| **Имя атрибута**:          | **`clean_users`**                           |     |
| **Содержится в:**          | [`contest`](contest.xml:contest.md) |     |
| **Тип значения:**          | *boolean*                                   |     |
| **Может отсутствовать:**   | *да*                                        |     |
| **Значение по умолчанию:** | true                                        |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Various contest's flags"*,
поле *"Allow pruning users?"*.

**Описание.** Если значение данного атрибута установлено в *true*
(значение по умолчанию), программа
[`clean-users`](clean-users.md) будет сканировать базу сдач
решений и вопросов судьям текущего турнира. Программа `clean-users`
выявляет пользователей, занесённых в базу пользователей, возможно
зарегистрировавшихся для участия в каком-либо турнире, но не сделавших
ни одной попытки сдачи решения и не задавших ни один вопрос. Такие
пользователи помечаются как кандидаты на удаление из базы пользователей.
Если значение данного атрибута установлено в *false*, программа
`clean-users` игнорирует данный турнир.
