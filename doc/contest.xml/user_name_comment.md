**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"extra HTML attributes for
CGI-programs"*, поле *"Additional comment for user name field"*.

**Описание.** Данные элемент позволяет задавать дополнительный текст,
который будет выводиться рядом с полем “User Name” регистрационной
анкеты. Этот текст может, например, содержать инструкции по выбору
названия команды. Текст выводится без изменений и может использовать
теги HTML, которые, однако, должны быть соответствующим образом
закодированы в XML файле.

**Пример.**

`<user_name_comment><u>Обратите внимание:</u>`  
` в названии команды сначала должно идти название школы, а затем`  
` (при желании, или если от школы выставляется более одной команды)`  
` - название команды. Например: "`  
` Школа 1234: Монстры программирования"`  
` или "Сборная школ 1, 2 и 3: Хакеры"`  
`</user_name_comment>`
