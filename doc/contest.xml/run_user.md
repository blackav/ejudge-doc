| **Имя элемента**:        | **`run_user`**                                    |
| **Содержится в:**        | [`contest`](contest.md)       |
| **Может содержать:**     | *нет*                                             |
| **Атрибуты:**            | *нет*                                             |
| **Тип содержимого:**     | регистрационное имя (login) пользователя в `Unix` |
| **Может отсутствовать:** | *да*                                              |
| **Может повторяться:**   | *нет*                                             |
| **Примечание:**          | В текущей версии `ejudge` игнорируется            |

**Описание.** Данный элемент позволяет указать регистрационное имя
пользователя в системе `Unix`, под которым будет запущена программа
[`run`](../run.md). Значение элемента предназначено для
использования в программа [`super-serve`](../super-serve.md), но в
текущей версии `ejudge` программа `super-serve` не позволяет работать
из-под пользователя `root` и не позволяет запускать другие программы
из-под произвольного пользователя, поэтому значение данного элемента
игнорируется.
