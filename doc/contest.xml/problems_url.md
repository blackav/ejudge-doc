**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Participation settings"*,
поле *"URL for the problemset"*.

**Описание.** Данный элемент позволяет задать URL для условий задач
турнира. Если значение элемента установлена, на личной странице
участника турнира, генерируемой CGI- программой
[`team`](../team.md), после начала турнира будет отображаться
ссылка на условия задач. Значение данного элемента используется без
изменений в атрибуте `href`.

**Пример.**

<problems_url>`/000100/problems.shtml`</problems_url>

Поскольку имя сервера не задано, данная ссылка будет вести на то же имя
сервера, которое было использовано при обращении к URL программы `team`.
