**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Registration settings"*, поле
*"Registration email sender (From: field)"*.

**Описание.** Данный элемент позволяет задать адрес отправителя (поле
`From`) электронного письма, посылаемого пользователю при регистрации в
системе. Это письмо посылается по адресу, который был задан
пользователем в регистрационной анкете и содержит регистрационное имя,
пароль пользователя, а также URL, для дальнейшей регистрации в системе.

Если данный элемент в конфигурационном файле турнира не задан,
используется значение элемента
[`register_email`](ejudge.xml:register_email.md)
конфигурационного файла [`ejudge.xml`](ejudge.xml.md). Если и
тот элемент не задан, используется адрес по умолчанию `USER`, где `USER`
— логин пользователя в `Linux`, от имени которого выполняется программа
[`userlist-server`](userlist-server.md). Полный почтовый адрес в
этом случае будет сформирован программой отсылки почты в `Linux` и
зависит от её настроек. Например, это может быть полный адрес DNS
компьютера, на котором работает `userlist-server`.

**Пример.**

<register_email>`register@supercontest.ru`</register_email>
