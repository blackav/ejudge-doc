Этот атрибут запрещает завершение турнира пользователем в том случае,
когда оно по умолчанию разрешено.

Завершение турнира по умолчанию разрешено в турнирах ограниченной
продолжительности.

Пример:

<contest id="auto" disable_user_finish="yes">  
</contest>

Поддерживается начиная с версии
[3.13.0](../изменения_в_версии_3.13.0.md).
