Данная конфигурационная переменная позволяет установить ограничение
размера виртуального адресного пространства специально для решений на
некотором языке программирования. Например,

`[problem]`  
`...`  
`max_vm_size = 32M`  
`lang_max_vm_size = "python3=64M"`  
`lang_max_vm_size = "kumir=128M"`

Устанавливает ограничение на размер адресного пространства в 32
мегабайта, но для программ на языке python3 ограничение устанавливается
в 64 мегабайта, а для программ на языке kumir — 128 мегабайт.
