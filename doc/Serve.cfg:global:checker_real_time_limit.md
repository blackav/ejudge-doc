|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`checker_real_time_limit`**           |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `run`                                   |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | `30`                                    |     |
| **Может повторяться:**     | *нет*                                   |     |

**Описание.** Данная конфигурационная переменная задаёт ограничение
астрономического времени на проверку результата работы тестируемой
программы на очередном тесте. Другими словами, эта переменная
ограничивает максимальное время работы проверяющей программы. В случае,
если максимальное время работы проверяющей программы превышено, проверка
программы участника завершается со статусом “Check failed” («Проверка не
удалась»). Значение 0 означает отсутствие ограничения времени.

**Пример.**

`checker_real_time_limit = 60`
