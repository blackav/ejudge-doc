В качестве параметра status_str могут передаваться следующие строки:

`None         не изменять статус регистрации`  
`"REMOVE"     удалить регистрацию пользователя на турнир`  
`"OK"         установить статус регистрации в OK`  
`"PENDING"    установить статус регистрации в PENDING`  
`"REJECTED"   установить статус регистрации в REJECTED`

Когда поле status_str входит в словать, возвращаемый запросом, его
значение может быть равно только "OK", "PENDING" или "REJECTED".
