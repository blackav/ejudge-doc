Этот атрибут разрешает завершение турнира пользователем в том случае,
когда оно по умолчанию запрещено.

Завершение турнира по умолчанию запрещено в турнирах неограниченной
продолжительности.

Пример:

<contest id="auto" enable_user_finish="yes">  
</contest>

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0 "wikilink").
