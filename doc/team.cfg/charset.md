|                            |                                                              |
|----------------------------|--------------------------------------------------------------|
| **Имя параметра:**         | **`charset`**                                                |
| **Тип содержимого:**       | *string*                                                     |
| **Может отсутствовать:**   | *да*                                                         |
| **Значение по умолчанию:** | устанавливается программой [configure](configure.md) |
| **Может повторяться:**     | *нет*                                                        |
| **Примечание:**            | использование не рекомендуется                               |

**Описание.** Данный параметр задаёт название кодировки, которое будет
помещаться в http-заголовок генерируемой страницы. Полная информация о
локализации системы `ejudge` приведена в разделе [Поддержка кодировок
символов](../поддержка_кодировок_символов.md).

Рекомендуется устанавливать кодировку по умолчанию с помощью опции
`--enable-charset` программы [configure](configure.md) при
компиляции системы `ejudge`. Поэтому использование данного параметра в
конфигурационном файле не рекомендуется.

**Пример.**

`charset = "koi8-r"`
