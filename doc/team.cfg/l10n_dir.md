| **Имя параметра:**         | **`l10n_dir`**                                                                                           |
| **Тип содержимого:**       | путь к каталогу                                                                                          |
| **Может отсутствовать:**   | *да*                                                                                                     |
| **Значение по умолчанию:** | устанавливается программой [configure](configure.md)                                             |
| **Может повторяться:**     | *нет*                                                                                                    |
| **Описание:**              | См. раздел [Включение поддержки языкового окружения](Включение_поддержки_языкового_окружения.md) |
| **Примечание:**            | использование не рекомендуется                                                                           |

Если параметр `enable_l10n` установлен в *true*, а параметр `l10n_dir`
не задан, локализация сообщений принудительно отключается.

Правильное значение параметра `enable_l10n` автоматически
устанавливается программой [configure](configure.md) по значению
опции `--prefix`. Кроме того, механизм локализации сообщений можно
отключить с помощью опции `--disable-nls`. Поэтому явное использование
параметров `l10n_dir` и `enable_l10n` не рекомендуется.

**Пример.**

`enable_l10n`  
`l10n_dir = "/usr/share/locale"`
