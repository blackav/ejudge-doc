**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings table attributes"*,
поле *"Ignore success time in penalty calculation"*.

Данная переменная действует только в режиме турнира *ACM*. Если
переменная установлена, то при вычислении штрафных баллов по задаче
будет игнорироваться время, прошедшее от начала турнира до успешной
сдачи. Переменная может использоваться в турнирах, где каждая задача
оценивается по системе зачёт/незачёт, и где время от начала турнира не
существенно.
