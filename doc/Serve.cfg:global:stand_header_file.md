| **Имя переменной**:        | **`stand_header_file`**                 |     |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *имя файла*                             |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *не установлено*                        |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"HTML header file for the standings"*.

**Описание.** Если данная конфигурационная переменная определена, то при
генерации таблицы текущего положения участников в начало генерируемого
файла будет помещаться текст, находящийся в файле-заголовке, полный путь
к которому задаётся в этой конфигурационной переменной. Содержимое
файла-заголовка считывается один раз при старте программы `serve`.

`stand_header_file = "/home/httpd/ssi/stand_head.shtml"`
