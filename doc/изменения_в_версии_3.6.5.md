### [ej-contests](ej-contests.md)

Номера запросов (параметр action) снова совместимы с номерами запросов
предыдущих версий (3.6.2 и ниже). Таким образом устранена регрессия при
обращении к ejudge по URL с указанием числового значения action.

Генерация .tex-протоколов поддерживается для турниров типа KIROV.
Временно отключен автоматический запуск latex и печать скомпилированных
протоколов. Отключена генерация протоколов тестирования программы в
.tex-протоколах. В следующих версиях ejudge включение/выключение этих
опций будет настраиваться из веб-интерфейса.
