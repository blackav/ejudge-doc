Глобальные опции позволяют изменять глобальные параметры настройки,
относящиеся ко всей системе ejudge. Эти параметры имеют корректные
значения по умолчанию, поэтому использовать глобальные опции не
рекомендуется.

`-f CONFIG-FILE`

Указать путь к альтернативному файлу конфигурации ejudge (ejudge.xml)
