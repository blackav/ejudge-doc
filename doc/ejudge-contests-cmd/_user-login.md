Команда user-login (можно использовать также имена команду team-login)
создаёт сессионный ключ для доступа к турниру с правами обычного
(непривилегированного) пользователя.

Использование:

`ejudge-contests-cmd CONTEST-ID user-login [OPTIONS] SESSION-FILE LOGIN [PASSWORD]`

В добавление к [стандартным опциям
команд](ejudge-contests-cmd:_COMMAND-OPTIONS.md) поддерживаются
следующие опции:

|     |                                                                                                             |
|-----|-------------------------------------------------------------------------------------------------------------|
| -f  | брать пароль не из командной строки, а из файла, имя которого указано в параметре PASSWORD командной строки |
| -p  | считать пароль с терминала. В этом случае параметр командной строки PASSWORD должен отсутствовать           |

Опции -f и -p не могут использоваться одновременно.

Если в качестве параметра SESSION-FILE указана строка `STDOUT`,
сессионный ключ будет записан не в файл, а на стандартный поток вывода.
