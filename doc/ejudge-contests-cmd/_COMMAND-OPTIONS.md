Опции команд позволяют изменить некоторые второстепенные параметры
выполнения команд. Опции команд общие для всех поддерживаемых команд.
Поддерживаются следующие опции.

`--ip=IP-ADDRESS`

Изменить IP-адрес, который будет использован для регистрации запроса в
журналах ejudge. По умолчанию используется IP-адрес 127.0.0.1
(localhost).

`--ssl`

Установить флаг использования протокола SSL, который будет использован
для регистрации запроса в журналах ejudge. По умолчанию флаг сброшен.

`--no-ssl`

Сбросить флаг использования протокола SSL. Опция добавлена для симметрии
с --ssl, так как флаг использования SSL сброшен по умолчанию.

`--session`

Указать, что значение сессионного ключа будет указано непосредственно в
параметрах команды вместо имени файла с сессионным ключем.

`--script-name=NAME`

Позволяет изменить имя программы, передаваемое серверу. По соглашению о
вызовах CGI-программ имя программы передаётся в переменной окружения
SCRIPT_NAME. По умолчанию имя программы — это полный путь к
new-server-cmd.

`--http-host=HOST`

Позволяет изменить имя компьютера, с которого посылается запрос. По
соглашению о вызовах CGI-программ этот параметр передаётся в переменной
окружения HTTP_HOST. По умолчанию он равен localhost.
