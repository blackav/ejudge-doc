**Описание.** В данном элементе хранится путь к «домашнему» каталогу
программы [`compile`](compile "wikilink"). Значение этого элемента
используется при создании новых турниров в программе
[`super-serve`](super-serve "wikilink").

Правильное значение этого элемента устанавливается на этапе
первоначальной настрой- ки системы `ejudge` после инсталляции с помощью
программы [`ejudge-setup`](ejudge-setup "wikilink"). После этого
изменять значение этого элемента не рекомендуется.
