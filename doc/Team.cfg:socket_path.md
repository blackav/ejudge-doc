| **Имя параметра:**         | **`socket_path`**                                            |
| **Тип содержимого:**       | путь к UNIX-сокету                                           |
| **Может отсутствовать:**   | *да*                                                         |
| **Значение по умолчанию:** | устанавливается программой [configure](configure.md) |
| **Может повторяться:**     | *нет*                                                        |
| **Примечание:**            | использование не рекомендуется                               |

**Описание.** Данный параметр задаёт путь к UNIX-сокету, который
используется для связи с программой-сервером базы пользователей
[userlist-server](userlist-server.md). В момент запуска
программы `team` сокет должен существовать.

Рекомендуется устанавливать путь к управляющему сокету при компиляции
системы `ejudge` с помощью опции `--enable-socket-path` скрипта
[configure](configure.md). В этом случае параметр `socket_path`
в конфигурационном файле использоваться не должен.

**Пример.**

<socket_path>`/var/ejudge/userlist-socket`</socket_path>
