CGI-программа `judge` предоставляет привилегированный (судейский)
интерфейс для проведения турнира. Только пользователи, у которых
установлена привилегия `JUDGE_LOGIN` для данного турнира могут
использовать программу `judge`. Программа `judge` позволяет
просматривать исходный код и результаты тестирования всех участников,
просматривать вопросы от участников и давать на них ответы,
просматривать таблицу текущих результатов турнира, когда она заморожена
для непривилегированных участников. Для каждого из этих действий
необходима отдельная привилегия, что позволяет тонко настраивать
возможности каждого пользователя. Аутентификация пользователей
выполняется с помощью запроса к программе
[`userlist-server`](userlist-server.md), а основные запросы
пользователей перенаправляются на обработку в программу
[`serve`](serve.md).
