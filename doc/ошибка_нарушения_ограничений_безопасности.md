Английское название: [Security
Violation](Security_Violation.md), [SE](SE.md)

Данная ошибка может диагностироваться при запуске тестируемой программы,
если она пытается выполнить недопустимый системный вызов, например,
создать сокет или создать процесс.

Диагностика данной ошибки включена только в режиме безопасного
выполнения программ
[`secure_run`](serve.cfg:global:secure_run.md) с включенной
опцией
[`detect_violations`](serve.cfg:global:detect_violations.md).
