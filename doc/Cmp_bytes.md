Стандартная проверяющая программа cmp_bytes сравнивает результат работы
тестируемой программы с эталонным ответом байт в байт. Никакого анализа
структуры файлов не выполняется. Эта проверяющая программа может
использоваться для сравнения в том числе и [бинарных
файлов](бинарный_файл.md).
