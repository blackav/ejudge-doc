**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Registration settings"*, поле
*"Virtual contest open date"*.

Элемент `open_time` позволяет задавать время открытия виртуального
турнира. Элемент применяется только для виртуальных турниров. Если
текущее время меньше времени открытия виртуального турнира, то
пользователь не сможет самостоятельно начать виртуальный турнир.

Время открытия виртуальных турниров задается в одном из [допустимых
форматов задания времени](формат_задания_времени "wikilink").
