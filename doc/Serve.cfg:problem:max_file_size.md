**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Maximum
file size"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Maximum file size"*.

Конфигурационная переменная поддерживается, начиная с версии 2.3.18.

Данная конфигурационная переменная позволяет устанавливать максимальный
размер файлов, которые могут создавать процессы. Например, если задано
значение

`max_file_size=64M`

тестируемые программы не смогут создавать выходные файлы больше 64
мегабайта.

Данная конфигурационная переменная наследуется из абстрактной задачи.
