Функции завершения работы проверяющей программы позволяют указать статус
завершения и некоторое диагностическое сообщение. Все функции никогда не
возвращают управление в основную программу.

|                                                  |                                                                                                                                             |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [`checker_OK`](libchecker:checker_OK.md) | завершить проверяющую программу со статусом OK                                                                                              |
| [`fatal`](libchecker:fatal.md)           | завершить проверяющую программу с заданным статусом                                                                                         |
| [`fatal_CF`](libchecker:fatal_CF.md)     | завершить проверяющую программу со статусом «[внутренняя ошибка проверки](внутренняя_ошибка_проверки.md)»                           |
| [`fatal_PE`](libchecker:fatal_PE.md)     | завершить проверяющую программу со статусом «[ошибка неправильного формата результата](ошибка_неправильного_формата_результата.md)» |
| [`fatal_WA`](libchecker:fatal_WA.md)     | завершить проверяющую программу со статусом «[неправильный ответ](неправильный_ответ.md)»                                           |
| [`fatal_read`](libchecker:fatal_read.md) | завершить проверяющую программу с [ошибкой неправильного формата файла](ошибка_неправильного_формата_файла.md)                      |
