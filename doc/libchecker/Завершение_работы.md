Функции завершения работы проверяющей программы позволяют указать статус
завершения и некоторое диагностическое сообщение. Все функции никогда не
возвращают управление в основную программу.

|                                                  |                                                                                                                                             |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [`checker_OK`](checker_OK.md) | завершить проверяющую программу со статусом OK                                                                                              |
| [`fatal`](fatal.md)           | завершить проверяющую программу с заданным статусом                                                                                         |
| [`fatal_CF`](fatal_CF.md)     | завершить проверяющую программу со статусом «[внутренняя ошибка проверки](../внутренняя_ошибка_проверки.md)»                           |
| [`fatal_PE`](fatal_PE.md)     | завершить проверяющую программу со статусом «[ошибка неправильного формата результата](../ошибка_неправильного_формата_результата.md)» |
| [`fatal_WA`](fatal_WA.md)     | завершить проверяющую программу со статусом «[неправильный ответ](../неправильный_ответ.md)»                                           |
| [`fatal_read`](fatal_read.md) | завершить проверяющую программу с [ошибкой неправильного формата файла](../ошибка_неправильного_формата_файла.md)                      |
