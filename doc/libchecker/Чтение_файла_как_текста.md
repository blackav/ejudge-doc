Функции данной группы позволяют считывать файл целиком в память и
работать с файлом в памяти.

|                                                                                              |                                                                                                                     |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [`checker_read_file`](checker_read_file.md)                               | считать файл целиком в память                                                                                       |
| [`checker_read_file_f`](checker_read_file_f.md)                           | считать файл целиком в память                                                                                       |
| [`checker_read_file_by_line`](checker_read_file_by_line.md)               | считать [текстовый файл](../текстовый_файл.md) построчно в память                                              |
| [`checker_read_file_by_line_f`](checker_read_file_by_line_f.md)           | считать [текстовый файл](../текстовый_файл.md) построчно в память                                              |
| [`checker_read_file_by_line_ex`](checker_read_file_by_line_ex.md)         | считать [текстовый файл](../текстовый_файл.md) построчно в память                                              |
| [`checker_read_line`](checker_read_line.md)                               | считать одну строку [текстового файла](../текстовый_файл.md) в память                                          |
| [`checker_read_line_ex`](checker_read_line_ex.md)                         | считать одну строку [текстового файла](../текстовый_файл.md) в память                                          |
| [`checker_read_buf_2`](checker_read_buf_2.md)                             | считать [элемент ввода](../элемент_ввода.md) из [текстового файла](../текстовый_файл.md) в буфер в памяти |
| [`checker_read_buf_ex`](checker_read_buf_ex.md)                           | считать [элемент ввода](../элемент_ввода.md) из [текстового файла](../текстовый_файл.md) в буфер в памяти |
| [`checker_eoln`](checker_eoln.md)                                         | проверить достижение конца строки в [текстовом файле](../текстовый_файл.md)                                    |
| [`checker_skip_eoln`](checker_skip_eoln.md)                               | пропустить остаток строки текста в [текстовом файле](../текстовый_файл.md)                                     |
| [`checker_skip_eoln_ex`](checker_skip_eoln_ex.md)                         | пропустить остаток строки текста в [текстовом файле](../текстовый_файл.md)                                     |
| [`checker_normalize_file`](checker_normalize_file.md)                     | удалить незначащие пробельные символы из считанного в память [текстового файла](../текстовый_файл.md)          |
| [`checker_normalize_spaces_in_file`](checker_normalize_spaces_in_file.md) | удалить повторяющиеся пробельные символы из считанного в память [текстового файла](../текстовый_файл.md)       |
| [`checker_normalize_line`](checker_normalize_line.md)                     | удалить незначащие пробельные символы из строки                                                                     |
|                                                                                              |                                                                                                                     |
