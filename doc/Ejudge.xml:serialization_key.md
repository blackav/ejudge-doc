**Описание.** В данном элементе хранится номер ключа семафора, который
используется программами [`compile`](compile.md) и
[`run`](run.md) для исключения одновременной компиляции или
запуска на выполнение тестируемых программ. Значение этого ключа
используется программой [`ejudge-setup`](ejudge-setup.md) для
генерации конфигурационного файла `compile.cfg` сервера компиляции, а
также программой [`super-serve`](super-serve.md) при генерации
конфигурационных файлов создаваемого турнира.

Значение ключа рекомендуется устанавливать при начальной настройке
системы `ejudge` с помощью программы
[`ejudge-setup`](ejudge-setup.md), а дальнейшее изменение не
рекомендуется.
