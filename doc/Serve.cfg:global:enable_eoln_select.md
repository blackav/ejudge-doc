**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Participants may select desired EOLN type"*.

Если данная глобальная конфигурационная переменная установлена в
положительное значение, участнику турнира предоставляется возможность
выбрать тип концов строк (Unix - \n или Dos - \r\n) в файлах, которые
будут подаваться на вход тестируемой программе.
