|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`stand_extra_legend`**                |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *string*                                |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | "" (пустая строка)                      |
| **Может повторяться:**     | *нет*                                   |

**Описание.** Данная конфигурационная переменная позволяет установить
название столбца дополнительной информации таблицы результатов турнира.
Столбец дополнительной информации активируется, если значение
конфигурационной переменной
[`stand_extra_format`](serve.cfg:global:stand_extra_format "wikilink")
не пусто.

**Пример.**

`stand_extra_legend = "Город, Страна"`
