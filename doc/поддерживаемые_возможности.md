Навигация: [Главная страница](Main_Page.md)/[Система
ejudge](Система_ejudge.md)/[Поддерживаемые
возможности](Поддерживаемые_возможности.md)

- Проведение турниров с автоматической проверкой задач по четырём
  системам: ACM, KIROV, OLYMPIAD, MOSCOW.
- Ограниченные и неограниченные по времени турниры.
- Поддержка виртуальных турниров.
- Одновременное проведение нескольких турниров.
- Автоматическая регистрация участников турнира. Модерируемая
  регистрация участников турнира.
- Возможность участия в нескольких турнирах под одним регистрационным
  именем.
- Разделение прав доступа к турнирам. Некоторый пользователь может быть
  администратором одного турнира и не иметь никаких привилегий в другом
  турнире.
- Многоязыковой интерфейс. Текущая версия поддерживает русский и
  английский языки.
- Защищённое исполнение программ (если установлен [патч к
  ядру](Патч_к_ядру_Linux.md)).
- Поддержка вариантных задач, когда под одним именем каждый участник
  получает свой вариант задачи.
- Веб-интерфейс администратора и участника турнира.
- Веб-интерфейс администратора турнира для создания новых турниров и
  редактирования настроек существующих турниров.
- Настраиваемый внешний вид.
- Экспорт журнала турнира в формате XML.
- Экспорт внутренних таблиц (участников, журнала турнира, результатов) в
  формате CSV (comma-separated values).
- [Доступ к серверам турниров из командной
  строки](new-server-cmd.md) (возможность написания скриптов для
  управления турнирами).
