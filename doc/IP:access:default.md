|                            |                                  |
|----------------------------|----------------------------------|
| **Имя атрибута**:          | **`default`**                    |
| **Содержится в:**          | [`access`](IP:access.md) |
|                            |                                  |
| **Тип значения:**          | *allow* или *deny*               |
| **Значение по умолчанию:** | *deny*                           |
| **Может отсутствовать:**   | *да*                             |

**Описание.** Данный атрибут задаёт права доступа в случае, если
IP-адрес клиента не перечислен в списке IP-адресов. Если атрибут
установлен в *allow*, такие IP-адреса считаются допустимыми, а если
атрибут установлен в *deny*, доступ с таких IP-адресов запрещается.
