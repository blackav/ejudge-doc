login - строка. Не должна быть None или пустой стокой. Должна быть
уникальной для всей базы пользователей. В принципе может содержать любые
печатные символы, но непривилегированным пользователям разрешается при
создании аккаунта использовать только латинские заглавные и строчные
буквы A-Z, a-z, цифры 0-9 и знаки - и \_. Не гарантируется, что
специальные HTML символы в логине будут отображаться на HTML страницах
корректно.
