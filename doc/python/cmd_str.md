Допустимые значения строки команды приведены ниже. Строка команды
определяет побитовую операцию текущих флагов регистрации и флагов
регистрации, указанных в параметре
[regflags](regflags.md).

`None      нет операции, параметр regflags игнорируется`  
`"NOP"     нет операции, параметр regflags игнорируется`  
`"SET"     установить указанные в regflags биты`  
`"CLEAR"   сбросить указанные в regflags биты`  
`"FLIP"    инвертировать указанные в regflags биты`
