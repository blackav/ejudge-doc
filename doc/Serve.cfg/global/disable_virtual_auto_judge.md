Переменная действует только для виртуальных турниров по системе
*OLYMPIAD*. Если значение переменной не равно 0, то сразу после
окончания виртуального турнира участника тестирование его решений
проводиться не будет. Тестирование решений должно запускаться
администратором турнира.
