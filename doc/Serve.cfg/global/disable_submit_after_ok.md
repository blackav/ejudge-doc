**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Disable submit of already solved problems"*.

Если эта переменная установлена в 1, то участникам запрещается посылка
уже сданных задач. Сданные задачи не появляются в списке задач,
доступных для сдачи.
