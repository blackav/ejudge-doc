Данный параметр позволяет задать максимальный суммарный размер исходного
кода программ и пользовательских данных всех [запусков на
пользовательских данных](Запуск_на_пользовательских_данных.md)
одного пользователя в одном турнире.

Значение параметра по умолчанию равно 1MiB.

<b>Пример:

`max_submit_total = 512K`

Поддерживается начиная с версии
[3.10.0](изменения_в_версии_3.10.0.md).
