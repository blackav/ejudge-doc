Данная переменная позволяет установить язык вывода сообщений
[проверяющих программ](проверяющие_программы.md), [оценивающих
программ](оценивающие_программы.md) и
[интеракторов](программы-интеракторы.md). Например, если язык
вывода установлен в `ru`, вместо вывода

`Answers differ: [1]: output: 19, corr: 20`

проверяющая программа выведет

`Несовпадение ответов: [1]: вывод: 19, эталон: 20`

Переменная может быть установлена в значения `russian`, `ru`, `1` (без
учета регистра букв), тогда всегда будет включен русский язык вывода.

Переменная может быть установлена в значение `user`, тогда язык вывода
проверяющих программ будет задаваться языком интерфейса пользователя в
момент посылки решения на проверку.
