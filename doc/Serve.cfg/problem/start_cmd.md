Данная конфигурационная переменная позволяет задать вспомогательную
программу для запуска решений участников на тестирование. Эта программа
может, например, подгружать решение участника в виде разделяемой
библиотеки или настраивать юнит-тестирование.

Вспомогательная программа копируется в каталог тестирования и
запускается из него с путем "./START-CMD". Далее в аргументах командной
строки ей передается имя тестируемой программы и все аргументы командной
строки для тестируемой программы. Кроме того до запуска вспомогательной
программы настраиваются все перенаправления ввода-вывода.

Поддерживается, начиная с версии
[3.2.0](Изменения_в_версии_3.2.0.md).
