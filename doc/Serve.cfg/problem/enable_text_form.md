**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Enable text
input form anyway"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Enable text
input form anyway"*.

При сдаче задач типа output-only разрешить ввод ответа в текстовом виде.

Для output-only задач поддерживается окно для ввода текстового ответа.
Таким образом, ответ может быть либо послан в файле в одном из
разрешенных форматов, либо введен непосредственно в окно ввода. Для
включения этой возможности должна быть установлена конфигурационная
переменная `enable_text_form` раздела описания задачи.

Конфигурационная переменная `enable_text_form` работает и для обычных
(`STANDARD`) задач на сдачу программ. Если значение переменной
установлено в 1, на странице сдачи отображается как окно для
непосредственного ввода текста программы, так и диалог для выбора файла
для сдачи. Пользователь может сдать программу на проверку либо
скопировав ее текст в окно, либо приложив файл. Сдать и текст программы,
и приложенный файл одновременно нельзя.
