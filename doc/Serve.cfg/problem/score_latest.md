Если данная переменная установлена в ненулевое значение, то в турнире по
системе *KIROV* вклад в общий результат участника будет давать последняя
посылка по этой задаче, а не лучшая. Опция может использоваться, когда в
турнирах по системе *KIROV* есть задачи, проверяемые в off-line режиме.
