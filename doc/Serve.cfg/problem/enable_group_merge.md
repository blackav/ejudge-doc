Если данный конфигурационный параметр установлен в положительное
значение, для текущей задачи включается [режим слияния подгрупп (правила
IOI)](Правила_IOI_расчета_баллов.md).

Параметр наследуется из абстрактной задачи.

Пример:

`[problem]`  
`enable_group_merge`

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0.md).
