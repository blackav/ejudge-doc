**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Convert PEs
to VAs"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Convert PEs to VAs"*.

Если данная конфигурационная переменная установлена в положительное
значение, то вердикт [ошибка неправильного формата
результата](ошибка_неправильного_формата_результата.md),
возвращаемый [проверяющей программой](проверяющие_программы.md)
автоматически преобразовывается в вердикт [неправильный
ответ](неправильный_ответ.md).

Значение переменной наследуется из абстрактной задачи, если не
определено в конкретной задаче.

Конфигурационная переменная поддерживается, начиная с версии
[2.3.26](изменения_в_версии_2.3.26.md) системы ejudge.
