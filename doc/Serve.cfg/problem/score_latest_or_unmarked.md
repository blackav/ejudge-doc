Если данная конфигурационная переменная установлена в 1, балл,
полученный по данной задаче, в турнире вычисляется следующим образом.

- Если среди посылок какого-либо пользователя по этой задаче
  присутствует хотя бы одна "отмеченная" (marked) посылка, то
  оценивается <b>последняя</b> из отмеченных посылок.
- Если среди посылок нет ни одной "отмеченной" посылки, то оценивается
  посылка, набравшая <b>максимальное</b> количество баллов.

Если переменная не определена для задачи, ее значение может быть
унаследовано из абстрактной задачи.

Переменная поддерживается начиная с версии
[2.3.18](изменения_в_версии_2.3.18.md).
