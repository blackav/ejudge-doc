|--------------------------|-------------------------------------------|
| **Имя переменной**:      | `checker_time_limit_ms`                   |
| **Содержится в:**        | [`problem`](serve.cfg:problem.md) |
| **Тип содержимого:**     | *integer*                                 |
| **Может отсутствовать:** | *да*                                      |
| **Наследуется:**         | *да*                                      |
| **Может повторяться:**   | *нет*                                     |

Данная конфигурационная переменная позволяет установить ограничение на
процессорное время для [проверяющих программ
(checkers)](Проверяющие_программы.md), [оценивающих программ
(valuers)](Оценивающие_программы.md) и [программ
инициализации](Программы_инициализации.md). Значение задаётся в
миллисекундах.

Если лимит процессорного времени превышается, диагностируется
[Внутренняя ошибка проверки](Внутренняя_ошибка_проверки.md).

Пример:

`[problem]`  
`# ...`  
`checker_time_limit_ms = 1000`

Поддерживается, начиная с версии
[3.9.0](изменения_в_версии_3.9.0.md).
