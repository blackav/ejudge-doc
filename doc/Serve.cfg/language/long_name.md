| **Имя переменной**:      | **`long_name`**                             |
| **Содержится в:**        | [`language`](serve.cfg:language.md) |
| **Используется:**        | `serve`                                     |
| **Тип содержимого:**     | *string*                                    |
| **Может отсутствовать:** | *да*                                        |
| **Может повторяться:**   | *нет*                                       |

**Описание.** Данная конфигурационная переменная устанавливает «длинное
имя» языка программирования. Длинное имя может содержать дополнительную
информацию о языке программирования, например, версию компилятора.
CGI-программа team использует значение этой переменной в меню выбора
языка для отсылки решения. Если эта конфигурационная переменная не
установлена, она автоматически устанавливается в значение
`"Language `<id>`"`, где <id> — это идентификатор задачи,
преобразованный в строку с помощью спецификации формата `%d`.

**Пример.**

`long_name = "GNU C 3.3.1"`
