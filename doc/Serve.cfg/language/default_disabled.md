языков](serve.cfg:language.md)/[`default_disabled`](Serve.cfg:language:default_disabled.md)

Если данный конфиграционный параметр установлен в положительное
значение, соответствующий язык не импортируется в турнир в режиме
[улучшенной поддержке серверов
компиляции](Улучшенная_поддержка_серверов_компиляции.md) при
указании директивы

`language_import = "enable all"`

Данный язык потребуется импортировать явно.

Данный конфигурационный параметр предназначен для использования в
конфигурационном файле сервера компиляции
[ej-compile](ej-compile.md). В конфигурационном файле турнира
[Serve.cfg](Serve.cfg.md) он игнорируется.

Пример:

`[language]`  
`id = 77`  
`short_name = "custom"`  
`long_name = "Custom"`  
`default_disabled`

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0.md).
