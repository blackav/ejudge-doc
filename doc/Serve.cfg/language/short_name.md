|--------------------------|---------------------------------------------|
| **Имя переменной**:      | **`short_name`**                            |
| **Содержится в:**        | [`language`](serve.cfg:language.md) |
| **Используется:**        | `serve`                                     |
| **Тип содержимого:**     | *string*                                    |
| **Может отсутствовать:** | *да*                                        |
| **Может повторяться:**   | *нет*                                       |

**Описание.** Данная конфигурационная переменная устанавливает «короткое
имя» языка программирования. Короткое имя используется в таблице решений
участников в CGI-программе `master`, в таблице решений в CGI-программе
`team`, а также для выбора языка программирования для отсылки решения.
Если эта конфигурационная переменная не установлена, она автоматически
устанавливается в значение `lang`<id>, где <id> — это идентификатор
задачи, преобразованный в строку с помощью спецификации формата `%d`.

**Пример.**

`short_name = "gcc"`
