Если данная конфигурационная переменная установлена в положительное
значение, то для задачи активируется режим [интеграции с системами
контроля версий](Интеграция_с_github/gitlab "wikilink").

Тем не менее, стандартный механизм сдачи решений на проверку остаётся
доступным.

Значение конфигурационной переменной наследуется из абстрактных задач.

Пример.

`[problem]`  
`# ...`  
`enable_vcs`

Поддерживается начиная с версии
[3.10.0](изменения_в_версии_3.10.0 "wikilink").
