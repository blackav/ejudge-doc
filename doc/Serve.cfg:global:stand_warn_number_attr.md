Если эта переменная установлена, в таблице текущих результатов
печатается столбец с общим количеством предупреждений для каждого
участника. С помощью глобальной конфигурационной переменной
`stand_warn_number_attr` можно задать HTML-атрибуты этого столбца,
например:

`stand_show_warn_number`  
`stand_warn_number_attr = ' width = "5%"'`

Обратите внимание на пробел в начале значения атрибута. Он необходим для
разделения тега <tt>

<td>

</tt> и значения атрибута.
