**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Participants cannot view failed test number"*.

Если эта переменная установлена в 1, то на странице участника не
отображается номер первого неправильного теста. Переменная работает
только для турниров по системе *ACM* или *MOSCOW*.
