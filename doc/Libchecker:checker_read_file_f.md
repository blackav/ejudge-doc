Функция checker_read_file_f — считать файл в память целиком.

`void checker_read_file_f(FILE *f, char **out, size_t *out_len);`

Функция считывает содержимое файла, задаваемого своим дескриптором
потока `f` в память. В переменную, на которую указывает параметр
`out_len`, будет записан размер считанного файла в байтах. В переменную,
на которую указывает `out`, будет записано содержимое файла. После
содержимого файла всегда дописывается нулевой байт, поэтому, если файл
[текстовый](текстовый_файл "wikilink"), то с переменной, в которую
возвращено его содержимое, можно работать как со строкой. Память для
содержимого файла выделяется на куче с помощью функций
[`xmalloc`](libchecker:xmalloc "wikilink") и
[`xrealloc`](libchecker:xrealloc "wikilink").

Функция не возвращает никакого значения. Если при чтении произошла
ошибка ввода, программа завершается, и диагностируется [внутренняя
ошибка проверки](внутренняя_ошибка_проверки "wikilink").
