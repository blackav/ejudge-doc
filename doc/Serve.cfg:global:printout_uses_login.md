Данная конфигурационная переменная модифицирует информацию, выносимую в
имя файла распечатки в режиме
[`disable_banner_page`](Serve.cfg:global:disable_banner_page "wikilink").
Если переменная не установлена, или установлена в неположительное
значение, имя файла распечатки будет содержать имя участника (поле
`name`). Если переменная установлена в положительное значение, имя файла
будет содержать логин участника (поле `login`).

Имя участника или логин участника полезны, чтобы идентифицировать
участника в аудитории и выдать ему его распечатку. В зависимости от
конфигурации турнира, созданных логинов и пр. в одних ситуациях проще
идентифицировать участников по их логину, в других ситуациях - по имени.

Данная конфигурационная переменная поддерживается, начиная с версии
[2.3.25](Изменения_в_версии_2.3.25 "wikilink").
