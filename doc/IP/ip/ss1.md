| **Имя атрибута**:          | **`ss1`**                                    |
| **Содержится в:**          | [`access`](IP:access.md)             |
| **Тип значения:**          | может принимать значения `yes`, `no` и `any` |
| **Значение по умолчанию:** | *any*                                        |
| **Может отсутствовать:**   | *да*                                         |

**Описание.** Данный атрибут задаёт тип соединения, к которому применимо
данное правило. Если значение атрибута равно `yes`, то данное правило
применимо только к соединениям по протоколу HTTPS, если значение
атрибута равно `no`, то данное правило применимо только к соединениям по
протоколу HTTP. Если значение атрибута равно `any`, или атрибут
отсутствует, то данное правило применимо к соединениям и по протоколу
HTTP, и по протоколу HTTPS.
