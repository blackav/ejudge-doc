**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"extra HTML attributes for
CGI-programs"*, поле *"Legend specification for users table"*.

**Описание.** Данный элемент позволяет управлять видом таблицы
пользователей, зарегистрированных на турнир. Элемент используется, когда
таблица зарегистрированных пользователей выводится на локальном языке
(`locale_id` не равно 0).

Элемент позволяет изменять вид заголовка таблицы. Изменять можно
заголовок столбцов таблицы со второй по предпоследнюю. Первая колонка
таблицы всегда называется `NN` (порядковый номер в списке), а
последняя - `Status` (или </tt>Статус</tt>) — статус регистрации
участника. Заголовки колонок отделяются друг от друга символом `|`
(вертикальная черта).

Например,

<users_table_legend>`Id|Имя участника|Учебное заведение|Факультет`</users_table_legend>

означает, что таблица зарегистрированных пользователей будет содержать
столбцы "NN "Id "Имя участника "Учебное заведение "Факультет "Статус".
Если значение данного элемента задано, обязательно должно быть задано
значение элемента
[`users_table_format`](contest.xml:users_table_format "wikilink"), при
этом оба элемента должны определять равное число стобцов.
