**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Valuer
works interactively"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Valuer
works interactively"*.

Если данная конфигурационная переменная установлена в положительное
значение, то [оценивающая программа](оценивающие_программы "wikilink")
запускается в интерактивном режиме.

Значение данной конфигурационной переменной наследуется из абстрактной
задачи, если не определено в конкретной задаче.

Переменная поддерживается, начиная с версии
[2.3.26](изменения_в_версии_2.3.26 "wikilink").
