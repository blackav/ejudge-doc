Программа `slice-userlist` генерирует «срез» базы пользователей по
заданному множеству турниров. В выходной XML-файл в формате базы
турниров попадают только те пользователи, которые были зарегистрированы
на указанное множество турниров. Все остальные пользователи,
присутствующие в базе пользователей, игнорируются. Полученный XML-файл
может использоваться как файл базы пользователей в другой конфигурации
системы `ejudge`. Программа работает с файлом базы пользователей
напрямую, поэтому перед выполнением этой программы система `ejudge`
должна быть полностью остановлена.
