Данный конфигурационный параметр языка программирования позволяет задать
программу, которая будет выполняться в самом конце тестирования
программы на каждом тесте на данном языке.

Эта программа запускается после запуска [программы
инициализации](Программы_инициализации "wikilink") с параметром `stop`,
то есть и после [проверяющей
программы](Проверяющие_программы "wikilink").

Программе очистки передаются различные [переменные
окружения](Переменные_окружения_для_запускаемых_программ "wikilink").

`[language]`  
`# ...`  
[`clean_up_cmd`](serve.cfg:language:clean_up_cmd "wikilink")` = "/opt/ejudge/libexec/ejudge/bin/ej-postgres-cleanup"`

Поддерживается начиная с версии
[3.12.0](изменения_в_версии_3.12.0 "wikilink").
