Переменная действует только для турниров по системе *OLYMPIAD* (в том
числе и виртуальных). Если значение переменной не равно 0, то сразу
после окончания турнира для участника на печать будет отправлен
пользовательский протокол экзамена. Для печати используется принтер,
заданный в поле `printer_name` данного пользователя. Если значение поля
`printer_name` не установлено, используется принтер по умолчанию.
