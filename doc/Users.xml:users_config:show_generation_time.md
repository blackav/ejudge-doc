[`show_generation_time`](users.xml:users_config:show_generation_time "wikilink")

|                            |                                                     |
|----------------------------|-----------------------------------------------------|
| **Имя атрибута**:          | **`show_generation_time`**                          |
| **Содержится в:**          | [`users_config`](users.xml:users_config "wikilink") |
| **Тип значения:**          | *boolean*                                           |
| **Значение по умолчанию:** | `false`                                             |
| **Может отсутствовать:**   | *да*                                                |

**Описание.** Если данный атрибут установлен в *true*, генерируемые
html-страницы будут содержать информацию о процессорном времени в
миллисекундах, затраченном на генерацию каждой страницы. Например, если
язык интерфейса — английский, сообщение будет иметь примерно следующий
вид:

`Page generation time: 37 msec`
