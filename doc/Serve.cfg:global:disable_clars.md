|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`disable_clars`**                     |     |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *boolean*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *false*                                 |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Clarification requests are disabled completely"*.

**Описание.** Если данная конфигурационная переменная установлена в
*true*, возможность участников турнира и судей и администраторов писать
сообщения друг другу отключена.

**Пример.**

`disable_clars`
