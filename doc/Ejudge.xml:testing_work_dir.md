**Описание.** Данный элемент хранит путь к рабочему каталогу для
тестирования программ. Значение этого элемента используется при создании
новых турниров в программе [`super-serve`](super-serve "wikilink").

Правильное значение этого элемента устанавливается на этапе
первоначальной настройки системы `ejudge` после инсталляции с помощью
программы [`ejudge-setup`](ejudge-setup "wikilink"). После этого
изменять значение этого элемента не рекомендуется.
