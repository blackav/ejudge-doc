**Описание.** В данном элементе хранится имя рабочего каталога программы
[`job-server`](job-server.md). Если значение элемента не задано,
то по умолчанию используется имя каталога `jwork`. Имя каталога может
быть как относительным путём, так и абсолютным. В случае относительного
пути абсолютный вычисляется с использованием значения элемента
`var_dir`.

Рекомендуемое значение данного элемента встроено по умолчанию, поэтому
явное использование не рекомендуется.
