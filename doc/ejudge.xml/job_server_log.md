**Описание.** В данном элементе хранится имя файла журнала программы
[`job-server`](../job-server.md). Файл журнала используется только
тогда, когда программа работает в режиме демона, т. е. запущена с опцией
`-D` командной строки. Если значение элемента не задано, то по умолчанию
используется имя файла `job-server.log`. Имя файла может быть как
относительным путём, так и абсолютным. В случае относительного пути
абсолютный вычисляется с использованием значения элемента `var_dir`.

Рекомендуемое значение данного элемента встроено по умолчанию, поэтому
явное использование не рекомендуется.
