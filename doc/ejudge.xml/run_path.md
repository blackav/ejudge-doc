| **Имя элемента**:          | **`run_path`**                                        |     |
| **Содержится в:**          | [`config`](config.md)              |     |
| **Может содержать:**       | *нет*                                                 |     |
| **Атрибуты:**              | *нет*                                                 |     |
| **Тип содержимого:**       | путь к программе [run](../run.md)                |     |
| **Значение по умолчанию:** | задается скриптом [`configure`](configure.md) |     |
| **Может отсутствовать:**   | *да*                                                  |     |
| **Может повторяться:**     | *нет*                                                 |     |
| **Используется в:**        | [`super-serve`](../super-serve.md)               |     |
| **Примечание:**            | явное использование не рекомендуется                  |     |

**Описание.** Данный элемент задаёт путь к программе
[`run`](../run.md) `системы ejudge`. При компиляции системы скрипт
[`configure`](configure.md) автоматически устанавливает
правильный путь к программе `serve`, основываясь на значении опции
`--prefix`. Поэтому явное использование этого тега в конфигурационном
файле не рекомендуется.

**Пример.**

<serve_path>`/usr/ejudge/bin/run`</serve_path>
