**Описание.** В данном элементе хранится имя каталога обмена программы
[`job-server`](../job-server.md), с помощью которого другие
программы системы `ejudge` передают запросы на выполнение заданий
программе [`job-server`](../job-server.md). Если значение элемента
не задано, то по умолчанию используется имя каталога `jspool`. Имя
каталога может быть как относительным путём, так и абсолютным. В случае
относительного пути абсолютный вычисляется с использованием значения
элемента `var_dir`.

Рекомендуемое значение данного элемента встроено по умолчанию, поэтому
явное использование не рекомендуется.
