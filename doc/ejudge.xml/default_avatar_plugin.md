Данный элемент позволяет задать имя плагина для [хранения
аватаров](../поддержка_аватаров.md) по умолчанию для всех турниров.

Пример:

<default_avatar_plugin>`mongo`</default_avatar_plugin>

Поддерживается с версии [3.7.0](../изменения_в_версии_3.7.0.md).
