**Описание.** Данный элемент позволяет задать кодировку символов,
которая будет использоваться программой
[`super-serve`](super-serve.md) при создании конфигурационных
файлов новых турниров.

Рекомендуется указывать кодировку символов по умолчанию при компиляции
системы `ejudge` с помощью опции `--enable-charset` скрипта
`[[configure}}`. В этом случае задание элемента `charset` в данном
конфигурационном файле не требуется.

**Пример.**

<charset>`koi8-r`</charset>
