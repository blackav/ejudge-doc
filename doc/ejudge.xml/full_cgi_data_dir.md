**Описание.** В данном элементе хранится полный путь к каталогу
конфигурационных файлов CGI-програм системы `ejudge`. Этот путь строится
в зависимости от значения полного пути к каталогу CGI-программ
веб-сервера, которое задаётся с помощью опции `--with-httpd-cgi-bin-dir`
скрипта [`configure`](configure.md), и от значения опции
`--enable-cgi-conf-dir` скрипта [`configure`](configure.md).
Так, если значение опции `--enable-cgi-conf-dir` является абсолютным
путём, то значение элемента `full_cgi_data_dir` будет совпадать со
значением опции.

Установка значения этого элемента выполняется программой
[`ejudge-setup`](../ejudge-setup.md) при начальной настройке
системы `ejudge` после инсталляции. Поэтому изменение значения этого
элемента при дальнейшей работе системы не рекомендуется.
