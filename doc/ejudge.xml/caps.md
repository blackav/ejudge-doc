| **Имя элемента**:        | **`caps`**                               |     |
| **Содержится в:**        | [`config`](config.md) |     |
| **Может содержать:**     | [`cap`](cap.md)       |     |
| **Атрибуты:**            | *нет*                                    |     |
| **Тип содержимого:**     | игнорируется                             |     |
| **Может отсутствовать:** | *да*                                     |     |
| **Может повторяться:**   | *да*                                     |     |

**Описание.** Данный элемент является элементом верхнего уровня списка
списков пол- номочий пользователей по отношению к базе данных
пользователей. Элемент внутри себя может содержать только элементы
`cap`. Если элемент `caps` встречается в конфигурационном файле
несколько раз, списки списков полномочий сливаются друг с другом.

**Пример.**

<caps>  
`  `<cap login="a"></cap>  
`  `<cap login="b"></cap>  
</caps>
