| **Имя элемента**:          | **`charset`**                                                               |
| **Содержится в:**          | [`serve_control_config`](serve_control_config.md) |
| **Может содержать:**       | *нет*                                                                       |
| **Атрибуты:**              | *нет*                                                                       |
| **Тип содержимого:**       | кодировка генерируемых HTML страниц                                         |
| **Значение по умолчанию:** | устанавливается [configure](configure.md)                           |
| **Может отсутствовать:**   | *да*                                                                        |
| **Может повторяться:**     | *нет*                                                                       |
| **Примечание:**            | явное использование не рекомендуется                                        |

**Описание.** Данный элемент позволяет задавать кодировку, которая будет
проставляться в заголовке всех генерируемых страниц. Обратите внимание,
что в текущей версии кодировка проставляется вне зависимости от языка
интерфейса.

Использование данного атрибута не рекомендуется. Предпочтительнее
устанавливать кодировку по умолчанию с помощью опции `--enable-charset`
скрипта [configure](configure.md).
