**Описание.** Данный элемент позволяет задавать ограничения на
IP-адреса, с которых производится обращение к CGI-программе
`serve-control`. Ограничения, заданные в этом конфигурационном файле,
проверяются до вывода регистрационной страницы и действуют на все
турниры, определённые в системе. Эти ограничения могут использоваться
для запрещения любого доступа к системе от «нежелательных» IP-адресов.
Каждый турнир может накладывать дополнительные ограничения. Они задаются
с помощью элемента
[`serve_control_access`](contest.xml:serve_control_access.md)
конфигурационного файла [`contest.xml`](contest.xml.md). См.
подробное описание ограничений IP-адреса в разделе [Ограничение доступа
по IP-адресам](Ограничение_доступа_по_IP-адресам.md).
