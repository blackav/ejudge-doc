| **Имя атрибута**:          | **`managed`**                               |     |
| **Содержится в:**          | [`contest`](contest.xml:contest "wikilink") |     |
| **Тип значения:**          | *boolean*                                   |     |
| **Может отсутствовать:**   | *да*                                        |     |
| **Значение по умолчанию:** | false                                       |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Various contest's flags"*,
поле *"Manage the contest server?"*.

**Описание.** Если значение этого атрибута установлено в true, программа
управления турниром [`serve`](serve "wikilink") ставится под управление
программы [`super-serve`](super-serve "wikilink"), которая следит
одновременно за несколькими турнирами и запускает программу `serve`, как
только поступает запрос соответствующему турниру. Если значение атрибута
установлено в *false*, данный турнир игнорируется программой
`super-serve`.
