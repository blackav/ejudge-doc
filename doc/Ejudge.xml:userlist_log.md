**Описание.** В данном элементе хранится имя файла журнала программы
[`userlist-server`](userlist-server "wikilink"). Файл журнала
используется только тогда, когда программа работает в режиме демона, т.
е. запущена с опцией `-D` командной строки. Если в этом случае файл
журнала не задан, программа выдает ошибку при запуске. Имя файла может
быть как относительным путём, так и абсолютным. В случае относительного
пути абсолютный вычисляется с использованием значения элемента
`var_dir`.

Рекомендуемое значение данного пути устанавливается автоматически на
этапе первона- чальной настройки системы программой
[`ejudge-setup`](ejudge-setup "wikilink"). После этого изменять значение
этого элемента не рекомендуется.

**Пример.**

<var_dir>`/home/judges/var`</contests>  
<userlist_log>`userlist.log`</userlist_log>

В данном случае файл журнала будет иметь полный путь

`/home/judges/var/userlist.log`
