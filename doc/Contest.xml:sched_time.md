**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Registration settings"*, поле
*"Contest start date"*.

Данный элемент позволяет задавать запланированное время старта турнира в
конфигурационном файле. При создании таблиц турнира запланированное
время старта копируется в таблицы турнира, после чего может изменяться
администратором турнира.
