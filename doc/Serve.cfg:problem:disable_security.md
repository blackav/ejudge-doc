**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Disable
security restrictions"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Disable
security restrictions"*.

Если установлена данная переменная, то для соответствующей задачи
отменяются ограничения безопасности.
