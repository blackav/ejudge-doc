Сервер ejudge может отправлять письма пользователям, например, для
подтверждения email при регистрации. Для этого используется программа
/usr/sbin/sendmail. Установите почтовый сервер postfix командой:

`# apt-get install postfix`

Настройте почтовый сервер, убедитесь, что отправляемые из командной
строки письма не попадают у получателей в папку "Спам". Вопросы
настройки postfix выходят за рамки данного руководства.
