**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Show the
problem after all results"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Show the
problem after all results"*.

Если эта переменная установлена в 1, то столбец данной задачи в таблице
текущих результатов будет отображаться после столбцов с суммарным баллом
и количеством решенных задач. Таким образом можно определять "задачи",
которые на самом деле будут содержать дополнительную информацию
(например, оценку за курс), проставляемую в таблицу текущих результатов.
