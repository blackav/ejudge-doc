|--------------------------|-------------------------------------------|
| **Имя переменной**:      | `checker_time_limit_ms`                   |
| **Содержится в:**        | [`problem`](serve.cfg:problem "wikilink") |
| **Тип содержимого:**     | *integer*                                 |
| **Может отсутствовать:** | *да*                                      |
| **Наследуется:**         | *да*                                      |
| **Может повторяться:**   | *нет*                                     |

Данная конфигурационная переменная позволяет установить ограничение на
процессорное время для [проверяющих программ
(checkers)](Проверяющие_программы "wikilink"), [оценивающих программ
(valuers)](Оценивающие_программы "wikilink") и [программ
инициализации](Программы_инициализации "wikilink"). Значение задаётся в
миллисекундах.

Если лимит процессорного времени превышается, диагностируется
[Внутренняя ошибка проверки](Внутренняя_ошибка_проверки "wikilink").

Пример:

`[problem]`  
`# ...`  
`checker_time_limit_ms = 1000`

Поддерживается, начиная с версии
[3.9.0](изменения_в_версии_3.9.0 "wikilink").
