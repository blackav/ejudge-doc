Если данная конфигурационная переменная установлена в положительное
значение, то в задаче будет поддерживаться [запуск на пользовательских
данных](Запуск_на_пользовательских_данных "wikilink").

Значение конфигурационной переменной наследуется из абстрактных задач.

Пример.

`[problem]`  
`# ...`  
`enable_user_input`

Поддерживается начиная с версии
[3.10.0](изменения_в_версии_3.10.0 "wikilink").
