**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Test
checker name"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Test checker name"*.

Данная конфигурационная переменная позволяет задать имя программы для
[проверки тестов](test_checkers "wikilink").

Программа проверки тестов запускается при выполнении операции "Check
contest settings" административного интерфейса для обычных задач, либо
при получении посылки для [задач на написание
тестов](Задача:tests "wikilink").

Данная переменная наследуется из абстрактной задачи. При наследовании
выполняется [форматная подстановка](Форматные_подстановки "wikilink").

Если значение данной переменной после наследования и форматных
подстановок представляет собой относительный путь, то дополнение
относительного пути до абсолютного зависит от режима работы турнира.
Если турнир работает в [альтернативной раскладке
файлов](альтернативная_раскладка_файлов "wikilink"), то путь к программе
отсчитывается от каталога задачи. Если турнир работает в стандартной
раскладке файлов, то путь к программе отсчитывается от каталога с
проверяющими программами.
