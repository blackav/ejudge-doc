Программа `clean-users` выполняет поиск по всем турнирам пользователей,
которые не сделали ни одной попытки сдачи решения и не задали ни одного
вопроса. Такие пользователи могут затем безболезненно быть удалены из
базы пользователей. Программа работает с файлами базы пользователей и
баз посылок турниров напрямую, поэтому перед выполнением этой программы
система `ejudge` должна быть полностью остановлена.
