задач](serve.cfg:problem.md)/[`score_tokenized`](Serve.cfg:problem:score_tokenized.md)

Данная переменная применима только в [турнирах с
токенами](турниры_с_токенами.md). Если переменная установлена в
положительное значение, то итоговый балл участника за эту задачу
вычисляется как максимальный балл от всех посылок, на которые
пользователь использовал токены.

Конфигурационная переменная может быть определена в абстрактной задаче и
будет наследоваться конкретными задачами.

Конфигурационная переменная поддерживается, начиная с версии
[3.3.0](изменения_в_версии_3.3.0.md).
