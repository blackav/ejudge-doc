Для создания бота для telegram можно следовать [инструкциям на сайте
telegram](https://core.telegram.org/bots).

Откройте приватный чат с ботом Botfather в клиенте Telegram.

Пошлите боту сообщение `/newbot`

После ответа бота введите имя для бота, например, EjudgeTestBot как
показано на рисунке ниже.

После второго ответа бота введите идентификатор бота, например,
EjudgeTestBot, как показано на рисунке ниже.

После этого вы получите сообщение, содержащее токен для доступа к Bot
API. На рисунке ниже этот токен равен
`275183432:AAHoM4qTSxTjTNV8ct0Z4pSL319oo5-JzPU`.

Здесь `275183432` - это идентификатор пользователя, а остальная часть
токена - секретный ключ, который не нужно пересылать или показывать
никому.
