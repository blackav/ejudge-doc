**Описание.** Данный элемент задаёт путь к каталогу, в котором хранятся
описания турниров [`contest.xml`](contest.xml "wikilink"). Каждое
описание турнира находится в отдельном файле с именем, равным
шестизначному номеру турнира, например, `000001.xml`.

Если при компиляции системы `ejudge` было указано значение данного пути
по умолчанию, то элемент `contests_dir` может отсутствовать. Для
указания пути к каталогам по умолчанию используется опция
`--enable-contests-dir` или опция `--enable-conf-dir` скрипта
[`configure`](configure "wikilink") при компиляции системы `ejudge`.

**Пример.**

<contests_dir>`/var/ejudge/contests`</contests_dir>
