**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Files and directories"*, поле
*"Advanced problem files layout"*.

Если данная глобальная конфигурационная переменная установлена в 1, в
турнире используется [альтернативная раскладка
файлов](альтернативная_раскладка_файлов.md).

По умолчанию значение данной переменной равно 0 (альтернативная
раскладка файлов отключена).
