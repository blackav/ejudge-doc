При запуске [программы
инициализации](Программы_инициализации "wikilink"),
[программы-интерактора](Программы-интеракторы "wikilink"), [оценивающей
программы](Оценивающие_программы "wikilink"), [проверяющей
программы](Проверяющие_программы "wikilink"), а также тестируемой
программы в режиме
[`enable_ejudge_env`](Serve.cfg:language:enable_ejudge_env "wikilink")
устанавливаются следующие переменные окружения.

|                            |                                                                         |                                                                                                                                                |
|----------------------------|-------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| Имя                        | Значение                                                                | Описание                                                                                                                                       |
| `EJUDGE`                   | `1`                                                                     | устанавливается всегда                                                                                                                         |
| `EJUDGE_CHECKER_TOKEN`     | `1`                                                                     | устанавливается в режиме [`enable_checker_token`](serve.cfg:problem:enable_checker_token "wikilink")                                           |
| `EJUDGE_CONTAINER`         | `1`                                                                     | устанавливается в режиме [`enable_container`](Serve.cfg:problem:enable_container "wikilink")                                                   |
| `EJUDGE_CONTEST_ID`        |                                                                         | идентификатор турнира (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))                            |
| `EJUDGE_CONTESTS_HOME_DIR` |                                                                         | каталог, в котором размещаются турниры (например, `/home/judges`)                                                                              |
| `EJUDGE_INTERACTIVE`       | `1`                                                                     | устанавливается в режиме [`interactive_valuer`](serve.cfg:problem:interactive_valuer "wikilink")                                               |
| `EJUDGE_LOCAL_DIR`         |                                                                         | каталог с рабочими файлами турнира (например, `/var/lib/ejudge`)                                                                               |
| `EJUDGE_LOCALE`            | значение [`checker_locale`](serve.cfg:global:checker_locale "wikilink") | язык вывода сообщений, например, `ru`                                                                                                          |
| `EJUDGE_MARKED`            | `1`                                                                     | устанавливается в режиме [`valuer_sets_marked`](Serve.cfg:problem:valuer_sets_marked "wikilink")                                               |
| `EJUDGE_MAX_SCORE`         |                                                                         | максимальный балл за данный тест в режиме [`scoring_checker`](serve.cfg:problem:scoring_checker "wikilink")                                    |
| `EJUDGE_PREFIX_DIR`        |                                                                         | каталог, в который проинсталлирована ejudge (например, `/opt/ejudge`)                                                                          |
| `EJUDGE_REJUDGE`           | `1`                                                                     | устанавливается при перетестировании (rejudge) посылки                                                                                         |
| `EJUDGE_RUN_ID`            |                                                                         | идентификатор посылки (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))                            |
| `EJUDGE_SCORING_CHECKER`   | `1`                                                                     | устанавливается в режиме [`scoring_checker`](serve.cfg:problem:scoring_checker "wikilink")                                                     |
| `EJUDGE_SERVER_BIN_PATH`   |                                                                         | каталог с серверными исполняемыми файлами (например, `/opt/ejudge/libexec/ejudge/bin`)                                                         |
| `EJUDGE_SOURCE_PATH`       |                                                                         | путь к файлу с исходным кодом в режиме [`enable_src_for_testing`](serve.cfg:problem:enable_src_for_testing "wikilink")                         |
| `EJUDGE_SUID_RUN`          | `1`                                                                     | устанавливается в режиме [`enable_suid_run`](serve.cfg:problem:enable_suid_run "wikilink") (устаревшее)                                        |
| `EJUDGE_SUPER_RUN_SERIAL`  |                                                                         | устанавливается при параллельном запуске [ej-super-run](ej-super-run "wikilink") для 1, 2, 3, ... (считая от 0) инстансов компонента           |
| `EJUDGE_TEST_COUNT`        |                                                                         | общее количество тестов в этой задаче (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))            |
| `EJUDGE_TESTLIB_MODE`      | `1`                                                                     | устанавливается в режиме [`enable_testlib_mode`](serve.cfg:problem:enable_testlib_mode "wikilink")                                             |
| `EJUDGE_TEST_NUM`          |                                                                         | номер теста (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))                                      |
| `EJUDGE_TEST_RANDOM_VALUE` |                                                                         | случайное 64-битное значение в 16-ричной записи. оно сохраняется для всех компонент, отрабатывающих для одной посылки на одном тесте           |
| `EJUDGE_USER_ID`           |                                                                         | идентификатор пользователя, который сдал посылку (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink")) |
| `EJUDGE_USER_LOGIN`        |                                                                         | логин пользователя (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))                               |
| `EJUDGE_USER_NAME`         |                                                                         | имя пользователя (только в режиме [`enable_extended_info`](serve.cfg:problem:enable_extended_info "wikilink"))                                 |
| `EJUDGE_USER_SCORE`        | `1`                                                                     | устанавливается в режиме [`separate_user_score`](Serve.cfg:global:separate_user_score "wikilink")                                              |
