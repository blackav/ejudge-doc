**Описание.** Данный элемент позволяет задать каталог конфигурационных
файлов и базы пользователей системы `ejudge`. Рекомендуется использовать
значение по умолчанию, задаваемое при компиляции системы `ejudge` с
помощью опции `--enable-contests-home-dir` скрипта
[`configure`](configure "wikilink"). Поэтому явное использование данного
элемента не рекомендуется.

**Пример.**

<config_dir>`/home/judges/data`</config_dir>
