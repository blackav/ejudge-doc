|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`autoupdate_standings`**              |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *boolean*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | `true`                                  |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле
*"Update standings automatically (except freeze time)"*.

**Описание.** Данная конфигурационная переменная управляет
автоматическим обновлением таблицы текущего положения участников
турнира. Если данная конфигурационная переменная установлена в *true*,
то таблицы текущего положения участников автоматически обновляются,
когда становятся известными результаты тестирования очередной программы,
а также в некоторых других случаях, за исключением времени «заморозки»
(см. конфигурационные переменные
[`board_fog_time`](serve.cfg:global:board_fog_time "wikilink"),
[`board_unfog_time`](serve.cfg:global:board_unfog_time "wikilink")).
Кроме этого администратор турнира имеет возможность в любой момент
обновить таблицу результатов. Если значение данной конфигурационной
переменной установлено в *false* с помощью строки

`autoupdate_standings = 0`

то таблица текущего положения никогда не обновляется автоматически.
Возможность обновлять таблицу текущих результатов с помощью
программы-администратора турнира сохраняется.
