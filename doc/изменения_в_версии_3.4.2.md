Исправления ошибок.

- корректная обработка `max_vm_size` и `max_stack_size` для java
  программ
- отображение ошибок сдачи программы на тестирование в привилегированном
  режиме
- checkbox обрабатываются корректно при редактировании посылки в
  административном интерфейсе
- ej-normalize корректно обрабатывает все спецсимволы
- добавлена поддержка языка GNU Assembler
- enable_max_stack_size обрабатывается корректно при редактировании
  настроек
- disable_team_clar обрабатывается корректно при редактировании настроек
- disable_user_database обрабатывается корректно при редактировании
  настроек
- отправка комментариев незарегистрированным администратором работает
  корректно
- условие конца файла проверяется после чтения ответа при проверке
  булевского ответа cmp_yesno
- номер первого непрошедшего теста отображается корректно в интерфейсе
  участника
