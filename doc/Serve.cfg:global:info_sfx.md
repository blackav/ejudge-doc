| **Используется:**          | `run`                                   |     |
| **Тип содержимого:**       | *string*                                |     |
| **Может отсутствовать:**   | *да*                                    |     |

**Описание.** Данная конфигурационная переменная устанавливает суффикс
файлов с дополнительной информацией о тестах. Эти файлы находятся в
подкаталогах каталога, задаваемого конфигурационной переменной
[`info_dir`](serve.cfg:global:info_dir "wikilink"). Формат файла с
дополнительной информацией о тесте описан в разделе
[.inf-файлы](.inf-файлы "wikilink").
