**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Registration settings"*, поле
*"Registration letter template file"*.

**Описание.** Данный элемент позволяет задать имя файла, в котором
хранится шаблон регистрационного письма, отсылаемого пользователю для
подтверждения регистрации. Если значение данного элемента не задано,
используется стандартная встроенная форма сообщения.

Значение элемента на самом деле представляет собой шаблон [форматной
подстановки](форматная_подстановка "wikilink"), в котором можно
использовать спецификации преобразования `%Ui, %Ul, %Ue, %Uz, %Vl, %Vu`.
Наиболее полезна спецификация `%Vl`, которая заменяется на номер
текущего языкового окружения — для выбора подходящего шаблона письма в
зависимости от языкового окружения, выбранного пользователем.

Если значением элемента является относительный путь, то полный путь к
файлу вычисляется относительно каталога конфигурационных файлов турнира,
определяемого с помощью элементов
[`root_dir`](contest.xml:root_dir "wikilink") или
[`conf_dir`](contest.xml:conf_dir "wikilink"). Рекомендуется всегда
использовать относительные пути.

Файл, указанный в данном элементе, также рассматривается как шаблон
[форматной подстановки](форматная_подстановка "wikilink"), в котором
можно использовать спецификации преобразования
`%Ui, %Ul, %Ue, %Uz, %Vl, %Vu`.

**Пример.**

<register_email_file>`regletter.txt`</register_email_file>
