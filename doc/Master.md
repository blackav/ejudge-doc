CGI-программа master предоставляет привилегированный административный
интерфейс для управления турниром. Только пользователи, у которых
установлена привилегия `MASTER_LOGIN` для данного турнира могут
использовать программу `master`. Помимо всех возможностей,
предоставляемых программой `judge`, программа `master` позволяет
начинать и заканчивать турнир, изменять его продолжительность,
пересуживать поступившие решения и выполнять много других действий.
Аутентификация пользователей выполняется с помощью запроса к программе
[`userlist-server`](userlist-server "wikilink"), а основные запросы
пользователей перенаправляются на обработку в программу
[`serve`](serve "wikilink").
