# ejudge-problem

Навигация: [Система ejudge](../система_ejudge.md)/[Использование](../использование.md)/[Общая архитектура системы](../общая_архитектура_системы.md)/[ejudge-problem](ejudge-problem.md)

Эта утилита командной строки предназначена для подготовки задачи к использованию
в турнирах.

Поддерживается начиная с версии [3.14.0](../change/изменения_в_версии_3.14.0.md).

Утилита выполняет различные действия в соответствии со спецификацией задачи.
Спецификация задачи по умолчанию находится в файле `ejproblem.xml`
в формате XML. Схема XML-файла напоминает схему XML-файла `problem.xml`
для задач, сформированных в системе Polygon, и, возможно, для простых задач
совместима с форматом `problem.xml`.

Использование:

```bash
ejudge-problem [GLOBAL-OPTIONS]... TOOL [TOOL-OPTIONS]
```

Поддерживаются следующие подкоманды TOOL:

| Подкоманда | Краткое описание |
|------------|------------------|
| `print-hash` | Вывести на стандартный поток вывода SHA256 от всех исходных файлов задачи |
| `print-makefile` | Вывести на стандартный поток вывода условный Makefile, с помощью которого выполняются все действия по подготовке задачи |
| `print-source-files` | Вывести на стандартный поток вывода список исходных файлов задачи |
| `print-topological` | Вывести на стандартный поток вывода порядок сборки файлов задачи |
| `print-build-info` | Вывести на стандартный поток вывода информацию о сборке задачи |
| `normalize` | Выполнить нормализацию тестов |
| `build` | Выполнить сборку задачи |
| `clean` | Удалить все сгенерированные файлы |
| `zip` | Собрать архив задачи, содержащий только исходные файлы |

## Конфигурационный файл

Конфигурационный файл называется `ejproblem.xml` и описывает правила для
подготовки задачи к тестированию.

*Исходными файлами* считаются все файлы, упомянутые в `ejproblem.xml`,
которые не являются результатом сборки из других файлов, и сам файл `ejproblem.xml`.

*Целевыми файлами* считаются все файлы, упомянутые в `ejproblem.xml`
и являющиеся результатом сборки.

Все остальные файлы в каталоге задачи игнорируются.
