# ejudge-edit-users

Навигация: [Система ejudge](../система_ejudge.md)/[Использование](../использование.md)/[Общая архитектура системы](../общая_архитектура_системы.md)/[ejudge-edit-users](ejudge-edit-users.md)

Программа `ejudge-edit-users` предоставляет пользовательский консольный
интерфейс для просмотра и редактирования базы пользователей системы
`ejudge`. Для выполнения запросов используется программа
[`ej-users`](../tool/ej-users.md).
