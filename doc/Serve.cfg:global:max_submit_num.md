Данный параметр позволяет задать максимальное количество [запусков на
пользовательских данных](Запуск_на_пользовательских_данных "wikilink")
одного пользователя в одном турнире.

Значение параметра по умолчанию равно 100.

<b>Пример:

`max_submit_num = 200`

Поддерживается начиная с версии
[3.10.0](изменения_в_версии_3.10.0 "wikilink").
