Ejudge поддерживает безопасный режим запуска программ, в котором
запускаемая программа не может выполнять никаких системных вызовов. Для
поддержки безопасного запуска необходима установка ядра со специальным
патчем. Собранные ядра с патчем (пакет kernel-image-std-cher) можно
скачать из дополнительных репозиториев, размещенных на
<ftp://ftp.altlinux.ru/pub/people/dk/>

В безопасном режиме могут быть проблемы с использованием
интерпретируемых языков программирования, таких как perl, python, ruby,
php.
