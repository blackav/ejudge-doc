**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Registration settings"*, поле
*"Virtual contest close date"*.

Элемент `close_time` позволяет задавать время закрытия виртуального
турнира. Элемент применяется только для виртуальных турниров. Если
текущее время больше времени закрытия виртуального турнира, то
пользователь не сможет самостоятельно начать виртуальный турнир. Если на
момент закрытия виртуального турнира есть текущие прохождения
виртуального турнира, они продолжаются до момента истечения их времени.

Время открытия виртуальных турниров задается в одном из [допустимых
форматов задания времени](формат_задания_времени.md).
