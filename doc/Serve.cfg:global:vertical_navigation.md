**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Place problem navigation vertically"*.

Данная переменная включает отображение закладок задач в одном столбце
страницы (вместо строки). Закладки в этом случае будут расположены
вертикально с правой стороны страницы.
