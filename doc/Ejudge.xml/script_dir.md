**Описание.** Значение данного элемента — это путь к каталогу в
инсталляции `ejudge`, в котором хранятся вспомогательные программы и
скрипты.

Корректное значение пути к каталогу вспомогательных программ
автоматически устанавливается скриптом
[`configure`](configure.md) по пути к корню инсталляции,
задаваемому с помощью опции `--prefix`. Явное использование элемента
`script_dir` не рекомендуется.
