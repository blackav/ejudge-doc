**Описание.** Данный элемент задаёт путь к каталогу, в котором хранятся
локализованные сообщения. Для локализации сообщений используется
`GNU gettext`.

При компиляции системы скрипт [`configure`](configure.md)
автоматически устанавливается правильный путь к каталогу сообщений,
основываясь на значении опции `--prefix`. Поэтому явное использование
этого тега в конфигурационном файле не рекомендуется.

**Пример.**

<l10n_dir>`/usr/share/ejudge/locale`</l10n_dir>
