| **Имя элемента**:        | **`register_email`**                            |     |
| **Содержится в:**        | [`config`](ejudge.xml:config "wikilink")        |     |
| **Может содержать:**     | *нет*                                           |     |
| **Атрибуты:**            | *нет*                                           |     |
| **Тип содержимого:**     | адрес электронной почты                         |     |
| **Может отсутствовать:** | *да*                                            |     |
| **Может повторяться:**   | *нет*                                           |     |
| **Используется в:**      | [`userlist-server`](userlist-server "wikilink") |     |

**Описание.** Данный элемент задаёт электронный адрес, от имени которого
рассылаются уведомления о регистрации новых пользователей. Если этот
элемент не определен ни в этом файле, ни в файле конфигурации турнира,
регистрационные сообщения рассылаются от имени пользователя Unix, под
которым запущена программа `userlist-server`. Значение, устанавливаемое
этим элементом может быть переопределено элементом
[`register_email`](contest.xml:register_email "wikilink")
конфигурационного файла турнира [`contest.xml`](contest.xml "wikilink").

**Пример.**

<register_email>`ejudge@olympiads.ru`</register_email>
