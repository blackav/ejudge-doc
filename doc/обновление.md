Для обновления версии ejudge достаточно:

- для прекомпилированной системы выполнить установку пакета (RPM или
  DEB);
- при компиляции из исходных кодов выполнить стандартные шаги configure,
  make, make install.

Все накопленные базы данных сохраняются и будут продолжаться
использоваться новой версией.
