| **Имя переменной**:        | **`prob_info_url`**                     |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *шаблон URL*                            |     |

| **Значение по умолчанию:** | *не установлено*                        |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"URL to view problem statement"*.

**Описание.** Данная конфигурационная переменная задаёт шаблон URL, по
которому находится текст условия задачи. Этот шаблон раскрывается в
момент генерации заголовка таблицы текущего положения участников в
турнире и формирует гиперссылку от идентификатора задачи к отдельной
странице с текстом этой задачи. Подробная информация о поддерживаемых
форматных преобразованиях дана в разделе [Форматные
подстановки](Форматные_подстановки "wikilink").

**Пример.**

`prob_info_url = "`[`http://www.contest.ru/problems/%lPs.shtml`](http://www.contest.ru/problems/%lPs.shtml)`"`
