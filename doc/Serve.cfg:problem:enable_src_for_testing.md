Если данная конфигурационная переменная установлена в положительное
значение, [проверяющей программе](Проверяющие_программы "wikilink"),
[интерактору](Программы-интеракторы "wikilink"), [программе
инициализации](Программы_инициализации "wikilink") и [оценивающей
программе](Оценивающие_программы "wikilink") будет доступен исходный код
посылки.

Путь к файлу с исходным кодом посылки находится в переменной окружения
`EJUDGE_SOURCE_PATH`. Язык программирования можно определить по суффиксу
имени файла.

Значение переменной наследуется из абстрактной задачи.

Пример.

`[problem]`  
`# ...`  
`enable_src_for_testing`

Поддерживается начиная с версии
[3.12.0](изменения_в_версии_3.12.0 "wikilink").
