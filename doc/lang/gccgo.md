Переменные окружения при компиляции

Хотя язык в ejudge называется gccgo, рекомендуется использовать
"официальный" компилятор go. На Fedora пакет называется `golang`. Кроме
того на Fedora может потребоваться выбрать "официальный" компилятор Go с
помощью команды

`alternatives --config go`

Для компиляции программы используется команда `go build`.

Компилятор Go использует статическую компоновку исполняемых файлов,
поэтому исполняемые файлы имеют размер несколько мегабайт даже для самых
простых программ. При определенных условиях можно использовать
конфигурационную переменную
[`enable_remote_cache`](../serve.cfg/global/enable_remote_cache.md)
для уменьшения количества передаваемых данных по сети.
