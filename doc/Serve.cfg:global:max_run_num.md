|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`max_run_num`**                       |     |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | 200                                     |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's quotas"*, поле
*"Maximum number of submits"*.

**Описание.** Данная конфигурационная переменная устанавливает
максимальное количество присылаемых на проверку программ от одного
участника. Количество присланных программ суммируется для каждого
участника по всем задачам. Если максимальное количество присылаемых
программ превышено, очередная программа будет отвергнута, а участнику
будет выдано сообщение об ошибке. Значение по умолчанию — 200, чего
должно быть достаточно для разумного использования.
