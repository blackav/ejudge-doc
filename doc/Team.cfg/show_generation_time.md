| **Имя параметра:**         | **`show_generation_time`**     |
| **Тип содержимого:**       | *boolean*                      |
| **Может отсутствовать:**   | *да*                           |
| **Значение по умолчанию:** | *false*                        |
| **Может повторяться:**     | *нет*                          |
| **Примечание:**            | использование не рекомендуется |

**Описание.** Если данный параметр установлен в *true*, в конце
генерируемой html-страницы добавляется информация о процессорном
времени, затраченном на её генерацию.

**Пример.**

`show_generation_time`
