Команда soft-update-stand обновляет таблицу текущих результатов турнира,
если только в данный момент не активна заморозка текущих результатов
турнира. По этой команде обновляются html-файлы текущих результатов
турнира, доступные не с помощью CGI-клиентов ejudge (то есть, файлы вида
<http://host/results/000001/standings.html>). Специальной команды для
обновления таблицы текущих результатов в CGI-клиентах системы ejudge не
требуется.

Для выполнения этой команды необходимы полномочия администратора
турнира.

Использование:

`ejudge-contests-cmd CONTEST-ID soft-update-stand [OPTIONS] SESSION-FILE`

Поддерживаются все [стандартные опции
команд](ejudge-contests-cmd:_COMMAND-OPTIONS.md). В частности,
если указана опция `--session`, то аргумент `SESSION-FILE` — это не
файл, в котором записан сессионный ключ, а сам сессионный ключ.
