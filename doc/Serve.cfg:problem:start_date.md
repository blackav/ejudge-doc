В этой переменной указывается астрономическое время, начиная с которого
данная задача становится доступной для сдачи.

Спецификация календарного штрафа может иметь одну из следующих форм:

`DATE TIME PENALTY  напр. 2004/10/10 12:00 -10`

или

`DATE PENALTY  напр. 2004/10/12 -20`

или

`PENALTY  напр. -30`

в последнем случае дата предполагается достаточно большой
(`2038/01/19` - то есть максимум, который позволяет точность типа
`time_t`).
