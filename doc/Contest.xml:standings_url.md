**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Participation settings"*,
поле *"URL for the current standings"*.

**Описание.** Данный элемент позволяет задать URL для страницы текущих
результатов турнира. Если значение элемента установлена, на личной
странице участника турнира, генерируемой CGI-программой
[`team`](team "wikilink"), после начала турнира будет отображаться
ссылка на таблицу текущих результатов турнира. Значение данного элемента
используется без изменений в атрибуте `href`. **Пример.**

<standings_url>`/000100/standings.shtml`</standings_url>

Поскольку имя сервера не задано, данная ссылка будет вести на то же имя
сервера, которое было использовано при обращении к URL программы `team`.
