Для установки системы ejudge на дистрибутивы ALT Linux необходимо
подключить стандартный репозиторий для вашего дистрибутива. После этого
установите пакет ejudge из репозитория командой:

`# apt-get update`  
`# apt-get install ejudge`

Рекомендуется использовать последние версии дистрибутив ALT Linux,
построенных на базе "Седьмой платформы". В более старых дистрибутивах
версия ejudge может также быть устаревшей. Также рекомендуется прочитать
замечания относительно особенностей тех или иных версий дистрибутивов.

## Восьмая платформа

Восьмая платформа - готовящиеся к выпуску в 2015-2016 года семейство
дистрибутивов ALT Linux.

## Седьмая платформа

"Седьмая платформа" - последний стабильный репозиторий ALT Linux, на
базе которого выпускались дистрибутивы в 2013-2015 годах, такие как:

- ALT Linux 7.0 Centaurus
- ALT Linux KDesktop 7.0
- ALT Linux Школьный 7.0
- Simply Linux 7.0

Стандартные репозитории для "Седьмой платформы" (**p7**) находятся по
адресу:

`rpm [p7] http://ftp.altlinux.org/pub/distributions/ALTLinux/p7/branch i586 classic`  
`rpm [p7] http://ftp.altlinux.org/pub/distributions/ALTLinux/p7/branch noarch classic`

Для данных дистрибутивов также существует репозиторий **t7**,
поддерживаемый сообществом ALT Linux. Версия ejudge в репозитории t7
может быть новее, чем в p7, поэтому для использования системы ejudge на
дистрибутивах "Седьмой платформы" рекомендуется подключение репозитория
t7:

`rpm http://ftp.altlinux.org/pub/distributions/ALTLinux/t7/branch i586 classic`  
`rpm http://ftp.altlinux.org/pub/distributions/ALTLinux/t7/branch noarch classic`

## Шестая платформа

"Шестая платформа" - устаревший стабильный репозиторий ALT Linux, на
базе которого выпускались дистрибутивы в 2011-2012 годах, такие как ALT
Linux 6.0 Кентавр, Simply, KDesktop, Школьный.

Дальнейшая поддержка ejudge в репозиториях p6 и t6 не гарантируется,
рекомендуется обновление до более новых дистрибутивов.

## Пятая платформа

"Пятая платформа" - устаревший стабильный репозиторий ALT Linux, на базе
которого выпускались дистрибутивы в 2009-2010 годах, такие как:

- Альт Линукс 5.0 Ковчег
- Альт Линукс 5.0 Школьный

Дальнейшая поддержка ejudge в репозиториях p5 и 5.1 не гарантируется,
рекомендуется обновление до более новых дистрибутивов.

## 4.1 и более старые дистрибутивы

Пакет ejudge в репозиториях 4.1 и 4.0 не обновлялся с 2009 года,
рекомендуется обновление дистрибутива до современного.
