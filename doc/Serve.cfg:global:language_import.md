Данный конфигурационный параметр позволяет задавать список языков
программирования, импортируемых в данный турнир в режиме [улучшенной
поддержки серверов
компиляции](Улучшенная_поддержка_серверов_компиляции.md).

Пример:

`enable_language_import`  
`language_import = "enable all"`

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0.md).
