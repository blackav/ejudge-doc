| **Содержится в:**        | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**        | `serve`                                 |     |
| **Тип содержимого:**     | *путь к каталогу*                       |     |
| **Может отсутствовать:** | *нет*                                   |     |
| **Может повторяться:**   | *нет*                                   |     |

**Описание.** Данная конфигурационная переменная задаёт путь к каталогу,
в котором находятся файлы описания турниров
[`contest.xml`](contest.xml "wikilink"). Каждое описание турнира
находится в отдельном файле с именем, равным шестизначному номеру
турнира, например, `000000.xml`. Используемое описание турнира
определяется с помощью конфигурационной переменной `contest_id`.

**Пример.**

`contests_dir = "/var/ejudge/contests"`
