**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"Internal XML log update interval (sec)"*.

С помощью переменной задается интервал времени в секундах между
генерацией XML-файла во внутреннем формате. Файл имеет следующий путь:

<contest_home>`/var/status/dir/internal.xml`
