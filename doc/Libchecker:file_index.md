Индекс файла — это целое значение от 0 до 2, которое является индексом в
массив f_arr трех дескрипторов потока проверяющей программы.

|     |                                                                                                                        |
|-----|------------------------------------------------------------------------------------------------------------------------|
| 0   | индекс файла с тестовыми данными, которые были поданы на вход тестируемой программе                                    |
| 1   | индекс файла с результатом работы тестируемой программы на данном тесте                                                |
| 2   | индекс файла с правильным ответом (или другой вспомогательной информацией для проверяющей программы) для данного теста |
