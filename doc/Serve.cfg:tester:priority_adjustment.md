| **Имя переменной**:        | **`priority_adjustment`**               |
| **Содержится в:**          | [`tester`](serve.cfg:tester "wikilink") |

| **Тип содержимого:**       | *integer*                               |

| **Наследуется:**           | *да*                                    |
| **Значение по умолчанию:** | `0`                                     |

| **Версия `ejudge`:**       | начиная с 2.1.29                        |

**Описание.** Данная конфигурационная переменная позволяет задать
поправку к приоритету тестирования для данного тестировщика. Значение
поправки будет прибавлено к базовому значению приоритета тестирования
(см. описание глобальной конфигурационной переменной
[`priority_adjustment`](serve.cfg:global:priority_adjustment "wikilink"))
при вычислении приоритета тестирования посылки. Положительное значение
поправки уменьшает приоритет, а отрицательное — увеличивает.

**Пример.**

`priority_adjustment = 2`
