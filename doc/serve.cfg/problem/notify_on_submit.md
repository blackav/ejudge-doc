Если данная конфигурационная переменная установлена в положительное
значение, то при завершения тестирования посылки по этой задаче в канал
в telegram будет отправлено сообщение о результате тестирования. Эта
конфигурационная переменная может использоваться для получения
уведомлений в задачах, которые требуют особенного внимания, например, в
задачах повышенной сложности которые не были пока сданы никем.

`notify_on_submit`

Конфигурационная переменная наследуется из абстрактной задачи.

Поддерживается с версии [3.6.3](../../изменения_в_версии_3.6.3.md).
