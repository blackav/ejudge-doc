# `exchange_dir`

Навигация: [Система ejudge](../../система_ejudge.md)/[Использование](../../использование.md)/[Конфигурационные файлы](../../конфигурационные_файлы.md)/[serve.cfg](../../serve.cfg.md)/[Конфигурационные параметры задач](../problem.md)/[`extra_src_dir`](exchange_dir.md)

Данный конфигурационный параметр позволяет задавать каталог для взаимодействия
с тестируемой программой. Этот каталог находится в том же каталоге,
в котором находится тестируемая программа, но права доступа к нему не изменяются
перед запуском тестируемой программы.

Если каталог обмена был создан перед запуском программы пользователем ejudge
и группой ejudge и правами 0777, они сохранятся во время работы тестируемой программы.
Это позволяет создавать временные файлы, например FIFO или сокетов.

Конфигурационный параметр должен быть допустимым для файловой системы
именем файла, то есть, в частности, не должен содержать символ `'/'`.

```text
[problem]
# ...
extra_src_dir = "lib"
```

Поддерживается начиная с версии [3.14.0](../../change/изменения_в_версии_3.14.0.md).
