Данная конфигурационная переменная позволяет установить балл за решение
задачи, которое получило предварительный (отображаемый участнику)
вердикт OK. Переменная имеет эффект, только если глобальная
конфигурационная переменная
[`separate_user_score`](../global/separate_user_score.md)
установлена в значение 1, то есть когда для каждой посылки отдельно
хранятся финальные баллы и предварительные баллы, а пользователю
отображаются только предварительные баллы.

Если переменная не определена для задачи, ее значение может быть
унаследовано из абстрактной задачи.

Переменная поддерживается начиная с версии
[2.3.18](../../изменения_в_версии_2.3.18.md).
