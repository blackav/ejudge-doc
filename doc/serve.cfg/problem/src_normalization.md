Данный конфигурационный параметр задаёт режимы преобразования файла с исходным кодом
сдаваемой программы. Например, если значение параметра установлено в `nl`,
в исходных текстах сдаваемых программ будет выполняться замена концов строк DOS (`\r\n`)
на концы строк Unix (`\n`). 

Эта опция полезна, когда для сдаваемых программ разрешено их копирование в форму ввода
[`enable_text_form`](enable_text_form.md). В этом случае браузер
передаёт текстовый файл в формате DOS. Это может вызвать многочисленные нарушения
стиля оформления программы если, например используется утилита clang-format.

```
[problem]
# ...
src_normalization = "nl"
```

Поддерживается начиная с версии [3.14.0](../../change/изменения_в_версии_3.14.0.md).
