Данная конфигурационная переменная позволяет установить имя стандартной
[оценивающей программы](../../оценивающие_программы.md) для данной
задачи. В настоящее время единственной стандартной программой оценивания
является [gvaluer](../../gvaluer.md). Установка этой конфигурационной
переменной позволяет не копировать gvaluer в каждый турнир, а
использовать программу, проинсталлированную в каталоги ejudge.

Значение переменной наследуется из абстрактной задачи, поэтому можно
установить значение один раз в абстрактной задаче, чтобы распространить
её действие на все конкретные задачи.

Поддерживается начиная с версии
[3.13.0](../../изменения_в_версии_3.13.0.md).
