**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Maximum
process count"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Maximum process
count"*.

Конфигурационная переменная поддерживается, начиная с версии 2.3.18.

Данная конфигурационная переменная позволяет устанавливать максимальное
число процессов, которые могут быть созданы пользователем.

Данная конфигурационная переменная наследуется из абстрактной задачи.
