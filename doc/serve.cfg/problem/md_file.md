Данный конфигурационный параметр позволяет задать имя файла, в котором находится
условие задачи в формате Markdown. Если задан и параметр `xml_file`, и параметр `md_file`,
используется `md_file`.

Конфигурационный параметр наследуется из абстрактной задачи.
Но если в конкретной задаче требуется отменить использование условий в Markdown-формате,
то можно в конкретной задаче установить значение параметра в пустую строку.

```
[problem]
# ...
md_file = "statement.md"
```

Поддерживается начиная с версии [3.14.0](../../change/изменения_в_версии_3.14.0.md).
