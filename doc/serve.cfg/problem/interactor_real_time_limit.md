Данная конфигурационная переменная позволяет установить ограничение
астрономического времени в секундах на работу
[программы-интеракторы](../../программы-интеракторы.md)
программы-интерактора. П умолчанию астрономическое время работы
интерактора не ограничено.

Конфигурационная переменная наследуется из абстрактной задачи.

Поддерживается с версии [3.6.2](../../изменения_в_версии_3.6.2.md).
