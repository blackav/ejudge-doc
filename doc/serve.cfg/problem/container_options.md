Эта конфигурационная переменная позволяет задать дополнительные опции
для программы контейнеризации
[ej-suid-container](ej-suid-container.md) для запуска
тестируемой программы на выполнение. Опция действует только когда
[разрешен запуск компиляторов в
контейнере](Изоляция_недоверенных_процессов_в_контейнерах.md).
Дополнительные опции дописываются в конец параметра опций строки запуска
контейнера.

<b>Внимание! Опции могут ослабить ограничения на запускаемые решения
участников турнира и привести к утечке информации! Не ослабляйте
ограничения, если это не является необходимым!</b>

См. также
[`lang_container_options`](serve.cfg:problem:lang_container_options.md).

Пример:

`[problem]`  
`# ...`  
`container_options = "s0mh"`

Поддерживается начиная с версии
[3.9.0](изменения_в_версии_3.9.0.md).
