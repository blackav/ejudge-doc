# `compile_error_penalty`

Навигация: [Система ejudge](../../система_ejudge.md)/[Использование](../../использование.md)/[Конфигурационные файлы](../../конфигурационные_файлы.md)/[serve.cfg](../../serve.cfg.md)/[Конфигурационные параметры задач](../problem.md)/[`compile_error_penalty`](compile_error_penalty.md)

Данная конфигурационная переменная позволяет задавать штраф за ошибки
компиляции. Переменная действует, если отключен режим
[`ignore_compile_errors`](ignore_compile_errors.md),

```text
compile_error_penalty = 1
```

Конфигурационная переменная наследуется из абстрактной задачи.

Поддерживается с версии [3.6.1](../../change/изменения_в_версии_3.6.1.md).
