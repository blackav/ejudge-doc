| **Имя переменной**:      | `checker_max_vm_size`                     |
| **Содержится в:**        | [`problem`](../problem.md) |
| **Тип содержимого:**     | *размер*                                  |
| **Может отсутствовать:** | *да*                                      |
| **Наследуется:**         | *да*                                      |
| **Может повторяться:**   | *нет*                                     |

Данная конфигурационная переменная позволяет установить ограничение на
размер виртуального адресного пространства [проверяющих программ
(checkers)](../../проверяющие_программы.md), [оценивающих программ
(valuers)](../../оценивающие_программы.md), [программ-интеракторов
(interactors)](../../программы-интеракторы.md) и [программ
инициализации](../../программы_инициализации.md).

Пример:

`[problem]`  
`# ...`  
`checker_max_vm_size = 256M`

Поддерживается, начиная с версии
[3.9.0](../../изменения_в_версии_3.9.0.md).
