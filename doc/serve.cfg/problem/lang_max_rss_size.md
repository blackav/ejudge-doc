Эта конфигурационная переменная позволяет задать ограничение на
максимальный размер использованной оперативной памяти (resident set
size - RSS) для решений данной задачи на определенных языках
программирования. Конфигурационная переменная может повторяться
несколько раз, указывая разные языки программирования.

Опция имеет эффект только тогда, когда [разрешен запуск компиляторов в
контейнере](../../изоляция_недоверенных_процессов_в_контейнерах.md). В
противном случае она игнорируется.

См. также [`max_rss_size`](max_rss_size.md),
[`max_rss_size`](../language/max_rss_size.md).

Пример.

`[problem]`  
`# ...`  
`lang_max_rss_size = "node=256M"`  
`lang_max_rss_size = "rust=64M"`

Поддерживается начиная с версии
[3.9.0](../../изменения_в_версии_3.9.0.md).
