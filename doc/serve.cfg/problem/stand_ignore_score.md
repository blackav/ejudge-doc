**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Ignore
problem score in standings"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Ignore
problem score in standings"*.

Если эта переменная установлена в 1, баллы за данную задачу не
добавляются к сумме всех баллов за турнир.
