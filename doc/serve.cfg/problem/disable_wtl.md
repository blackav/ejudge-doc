**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Convert
WTLs to TLs"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Convert WTLs to TLs"*.

Если данная конфигурационная переменная установлена в положительное
значение, то вердикт [ошибка превышения лимита реального
времени](../../ошибка_превышения_лимита_реального_времени.md),
возвращаемый [проверяющей программой](../../проверяющие_программы.md)
автоматически преобразовывается в вердикт [ошибка превышения лимита
времени](../../ошибка_превышения_лимита_времени.md).

Значение переменной наследуется из абстрактной задачи, если не
определено в конкретной задаче.

Конфигурационная переменная поддерживается, начиная с версии
[2.3.26](../../изменения_в_версии_2.3.26.md) системы ejudge.
