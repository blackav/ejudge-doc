**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Valuer sets
\_marked\_ flag"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Valuer sets \_marked\_
flag"*.

Если данная конфигурационная переменная установлена в 1, [оценивающая
программа](../../оценивающие_программы.md) (valuer) для данной задачи
должен дополнительно выдать флаг "marked" для данной посылки.
Оценивающая программа таким образом должна напечатать на стандартный
поток вывода два числа: балл за данную посылку и значение флага "marked"
данной посылки.

Флаг "marked" для посылок далее может использоваться в выражении фильтра
административного интерфейса.

Значение данной конфигурационной переменной по умолчанию равно 0, то
есть оценивающая программа должна только вывести балл за данную посылку.

Данная конфигурационная переменная наследуется из абстрактной задачи.
