**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Use AC
status instead of OK"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Use AC
status instead of OK"*.

Если эта конфигурационная переменная установлена, то вместо статуса
[OK](../../verdict/OK.md) для решений, проходящих все тесты, используется
статус [Accepted for Testing](../../verdict/accepted_for_testing.md).

Начиная с версии [2.3.25](../../change/изменения_в_версии_2.3.25.md)
используется статус [Pending Review](../../pending_Review.md).
