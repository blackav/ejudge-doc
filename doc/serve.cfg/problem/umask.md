Данная конфигурационная переменная позволяет задать параметр
[umask](https://ru.wikipedia.org/wiki/Umask) для запускаемого процесса.
При использовании в [безопасном режиме](безопасном_режиме.md)
значение переменной может не оказать влияние на тестируемые программы,
так как создание новых файлов запрещено.

Конфигурационная переменная может быть определена в абстрактной задаче и
будет наследоваться конкретными задачами.

Значение записывается в восьмеричном виде, например:

`umask = 022`

Конфигурационная переменная поддерживается, начиная с версии
[3.2.1](../../изменения_в_версии_3.2.1.md).
