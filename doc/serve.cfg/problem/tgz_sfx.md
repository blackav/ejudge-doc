**Описание.** Данная конфигурационная переменная устанавливает суффикс
файлов архивов рабочего каталога для тестирования программ. Если
неабстрактная задача не устанавливает непосредственно данную переменную,
но наследует абстрактную задачу, устанавливающую эту конфигурационную
переменную, используется значение из описания абстрактной задачи. Если
после этого значение конфигурационной переменной `tgz_sfx` всё ещё не
определено, используется значение глобальной конфигурационной переменной
[`tgz_sfx`](../global/tgz_sfx.md), которая по умолчанию
равна `.tgz`. Таким образом можно считать, что файлы с архивами рабочего
каталога имеют по умолчанию суффикс `.tgz`.

Если в конфигурации задачи определены и конфигурационная переменная
[`tgz_pat`](tgz_pat.md), и конфигурационная
переменная [`tgz_sfx`](tgz_sfx.md),
используется значение переменной
[`tgz_pat`](tgz_pat.md).
