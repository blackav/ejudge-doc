задач](../problem.md)/[`require_any`](require_any.md)

Если эта конфигурационная переменная установлена в 1, то для того чтобы
сделать эту задачу доступной для решения участник должен решить хотя бы
одну из задач, перечисленных в списке
[`require`](require.md).

Переменная может задаваться в абстрактной задаче и будет унаследована.

Например, если для некоторой задачи задана следующая конфигурация:

`require = "A"`  
`require = "B"`  
`require_any`

то эта задача открывается, если участник успешно решил либо задачу A,
либо задачу B. Если бы параметр `require_any` не был задан, требовалось
бы решить и задачу A, и задачу B.
