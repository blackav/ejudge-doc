Если эта конфигурационная переменная установлена в 1, и задача является
[вариантной](variant_num.md), то вариант
задачи, выдаваемый участнику, будет назначаться автоматически, если он
не был до этого назначен вручную. Вариант назначается участнику при
первом открытии им вариантной задачи. Назначенный вариант автоматически
сохраняется в [файл отображения
вариантов](../global/variant_map_file.md).

Конфигурационная переменная игнорируется для обычных (не вариантных)
задач. Конфигурационная переменная может быть определена в абстрактной
задаче и будет унаследована конкретными задачами, если в них не будет
переопределена.

Данная конфигурационная переменная поддерживается, начиная с версии
[3.2](../../изменения_в_версии_3.2.0.md)
