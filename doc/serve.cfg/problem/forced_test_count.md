Данный параметр конфигурации позволяет ограничить количество тестов, используемых
для тестирования задачи, указанным в параметре количеством.
Например, если в задаче задано 23 теста, но параметр `forced_test_count` равен 3,
для тестирования будут использованы только 3 теста.

Конфигурационный параметр **не наследуется** из абстрактной задачи.

```
[problem]
# ...
forced_test_count = 3
```

Поддерживается начиная с версии [3.14.0](../../change/изменения_в_версии_3.14.0.md).
