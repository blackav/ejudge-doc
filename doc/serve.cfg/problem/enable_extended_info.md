Если эта конфигурационная переменная установлена в 1, то в
[программу-интерактор](../../программы-интеракторы.md) и [проверяющую
программу](../../проверяющие_программы.md) с помощью переменных
окружения передается дополнительная информация о тестируемой программе.

|                      |                                                               |
|----------------------|---------------------------------------------------------------|
| Переменная окружения | Описание                                                      |
| `EJUDGE_USER_ID`     | Идентификатор пользователя ejudge — положительное целое число |
| `EJUDGE_CONTEST_ID`  | Идентификатор турнира ejudge — положительное целое число      |
| `EJUDGE_RUN_ID`      | Идентификатор посылки — неотрицательное целое число           |
| `EJUDGE_TEST_NUM`    | Номер теста — положительное целое число                       |
| `EJUDGE_USER_LOGIN`  | Логин пользователя                                            |
| `EJUDGE_USER_NAME`   | Имя пользователя                                              |

Значение данной конфигурационной переменной наследуется из абстрактной
задачи, если оно определено в абстрактной задаче и не переопределено в
конкретной задаче.

Поддерживается начиная с версии
[3.8.0](../../изменения_в_версии_3.8.0.md). В версии
[3.9.0](../../изменения_в_версии_3.9.0.md) дополнительные переменные
окружения передаются и в [проверяющую
программу](../../проверяющие_программы.md).
