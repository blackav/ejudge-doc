**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Disabled
languages"*.

**Описание.** Данная конфигурационная переменная позволяет указать языки
программирования, которые не могут использоваться для посылок по данной
задаче. Каждый раз указывается только одно короткое имя
([`short_name`](serve.cfg:problem:short_name.md)) языка
программирования, но переменная `disable_language` может быть
использована несколько раз в описании одной задачи.

**Пример.**

`disable_language = "gcc"`  
`disable_language = "g++"`

Запрещает для некоторой задачи использование языков с коротким именем
([`short_name`](serve.cfg:problem:short_name.md)) `gcc` и `g++`.
