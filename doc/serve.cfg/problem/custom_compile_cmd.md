# `custom_compile_cmd`

Навигация: [Система ejudge](../../система_ejudge.md)/[Использование](../../использование.md)/[Конфигурационные файлы](../../конфигурационные_файлы.md)/[serve.cfg](../../serve.cfg.md)/[Конфигурационные параметры задач](../problem.md)/[`custom_compile_cmd`](custom_compile_cmd.md)

Данный конфигурационый параметр позволяет задавать имя скрипта
компиляции для [специальной компиляции](../../специальная_компиляция.md).

В турнир должен быть добавлен язык программирования `custom`. Задача, у
которой задан параметр custom_compile_cmd может быть сдана только на
этом языке программирования. Все остальные языки, разрешенные в турнире,
для нее автоматически запрещаются. Пример.

```text
[problem]
# ...
custom_compile_cmd = "compile"
custom_lang_name = "bash"
```

Поддерживается начиная с версии [3.10.2](../../change/изменения_в_версии_3.10.2.md).
