Переменная работает только для турниров *OLYMPIAD* и позволяет задать
минимальное количество тестов, которое должна пройти программа при
предварительном тестировании, чтобы быть принятой на проверку.
Предположем, что всего предварительных тестов 3, из них для принятия
задачи на проверку достаточно только одного. Тогда раздел описания
задачи может содержать следующие строки:

`tests_to_accept = 3`  
`min_tests_to_accept = 1`

Значение переменной наследуется от абстрактной задачи.
