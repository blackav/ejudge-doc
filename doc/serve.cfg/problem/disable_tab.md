**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Disable
problem tab"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Disable problem tab"*.

Если значение этой переменной не равно 0, для данной задачи не будет
отображаться закладка (в режиме `problem_navigation`).
