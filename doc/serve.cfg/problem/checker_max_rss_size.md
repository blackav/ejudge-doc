Данная конфигурационная переменная позволяет установить ограничение на
размер использованной оперативной памяти (resident set size - RSS)
[проверяющих программ (checkers)](../../проверяющие_программы.md),
[оценивающих программ (valuers)](../../оценивающие_программы.md),
[программ-интеракторов (interactors)](../../программы-интеракторы.md)
и [программ инициализации](../../программы_инициализации.md).

Ограничение RSS действует только при выполнении программ в контейнерах.
Поскольку указанные выше программы не запускаются в контейнере,
ограничение на RSS не проверяется.

Пример:

`[problem]`  
`# ...`  
`checker_max_rss_size = 256M`

Поддерживается, начиная с версии
[3.9.0](../../изменения_в_версии_3.9.0.md).
