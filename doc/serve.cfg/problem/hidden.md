|                            |                                           |
|----------------------------|-------------------------------------------|
| **Имя переменной**:        | **`hidden`**                              |
| **Содержится в:**          | [`problem`](../problem.md) |
| **Используется:**          | `serve`                                   |
| **Тип содержимого:**       | *boolean*                                 |
| **Может отсутствовать:**   | *да*                                      |
| **Наследуется:**           | *да*                                      |
| **Значение по умолчанию:** | *false*                                   |
| **Может повторяться:**     | *нет*                                     |
| **Версия `ejudge`:**       | начиная с 2.1.26.                         |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Do not show
this problem in standings"*, либо страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Concrete problems"*, поле *"Do not show
this problem in standings"*.

**Описание.** Если данная конфигурационная переменная описания некоторой
задачи установлена в *true*, эта задача помечается как «скрытая».
Скрытые задачи не отображаются в таблице текущих результатов турнира,
однако доступны участникам турнира для сдачи решений.

Если в описании неабстрактной задачи данная конфигурационная переменная
не была определена, используется значение переменной из абстрактной
задачи, наследуемой данной задачей. Если после этого значение данной
конфигурационной переменной не определено, используется значение по
умолчанию *false*.
