Если данный конфигурационный параметр языка программирования установлен
в положительное значение, при запуске решений на данном языке
программирования будут устанавливаться все [переменные
окружения](Переменные_окружения_для_запускаемых_программ.md),
которые устанавливаются при запуске служебных программ, таких как
[проверяющие программы](Проверяющие_программы.md).

Этот конфигурационный параметр используется сейчас для
[`PostgresQL`](Lang:postgres.md).

`[language]`  
`# ...`  
`enable_ejudge_env`

Поддерживается начиная с версии
[3.12.0](изменения_в_версии_3.12.0.md).
