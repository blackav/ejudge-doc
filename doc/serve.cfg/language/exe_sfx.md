| **Имя переменной**:        | **`exe_sfx`**                               |
| **Содержится в:**          | [`language`](../language.md) |
| **Используется:**          | `serve`                                     |
| **Тип содержимого:**       | *string*                                    |
| **Может отсутствовать:**   | *да*                                        |
| **Значение по умолчанию:** | "" (пустая строка)                          |
| **Может повторяться:**     | *нет*                                       |

**Описание.** Данная конфигурационная переменная устанавливает суффикс
исполняемых файлов, получаемых в результате компиляции данным
компилятором. Как правило, суффикс исполняемых файлов зависит от
архитектуры. Например, исполняемые файлы для `Linux` обычно не имеют
никакого суффикса, а исполняемые файлы для `DOS` имеют суффикс `.exe`.

**Пример.**

`exe_sfx = ".exe"`
