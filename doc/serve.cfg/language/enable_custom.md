Если данный конфигурационный параметр установлен, текущий язык
программирования является общим языком для поддержки [специальной
компиляции](специальная_компиляция.md). В конфигурационном файле
допускается единственный язык с установленным флагом `enable_custom`.

Поддерживается начиная с версии
[3.10.2](изменения_в_версии_3.10.2.md).
