|                            |                                             |
|----------------------------|---------------------------------------------|
| **Имя переменной**:        | **`compile_real_time_limit`**               |
| **Содержится в:**          | [`language`](../language.md) |
| **Используется:**          | `serve`                                     |
| **Тип содержимого:**       | *boolean*                                   |
| **Может отсутствовать:**   | *да*                                        |
| **Значение по умолчанию:** | *false*                                     |
| **Может повторяться:**     | *нет*                                       |
| **Версия `ejudge`:**       | начиная с 2.1.27                            |

**Описание.** Данная конфигурационная переменная задаёт ограничение
астрономического времени на компиляцию программы участника. В случае,
если максимальное время компиляции превышено, проверка программы
участника завершается со статусом “Check failed” («Проверка не
удалась»). Значение 0 означает отсутствие ограничения времени. Если
данная конфигурационная переменная не установлена, используется значение
глобальной конфигурационной переменной
[`compile_real_time_limit`](../global/compile_real_time_limit.md).

**Пример.**

`compile_real_time_limit = 60`
