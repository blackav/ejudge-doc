|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`diff_path`**                         |
| **Содержится в:**          | [`global`](../global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *путь к программе*                      |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | `/usr/bin/diff`                         |
| **Может повторяться:**     | *нет*                                   |
| **Версия `ejudge`:**       | начиная с 2.1.27.                       |

**Описание.** Данная конфигурационная переменная позволяет задать путь к
программе `diff`, которая используется для сравнения текстов двух
посылок.

**Пример.**

`lpr_path = "/usr/local/bin/diff"`
