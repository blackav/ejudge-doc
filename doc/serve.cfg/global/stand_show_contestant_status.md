С помощью глобальной переменной `stand_show_contestant_status` можно
включить отображение столбца статуса участников. Для каждого участника
будет напечатан его статус, как задано в переменной
[`contestant_status_legend`](serve.cfg:problem:contestant_status_legend.md).
