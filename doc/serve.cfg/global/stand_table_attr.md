**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings table attributes"*,
поле *"HTML attributes for standings table"*.

**Описание.** Данная конфигурационная переменная позволяет определить
дополнительные атрибуты элемента `<table>` таблицы текущих результатов.
Значение данной переменной и элемент `table` не разделяются
дополнительными пробелами, поэтому значение данной переменной должно
начинаться с пробельного символа.

**Пример.**

`stand_table_attr = " width=100%"`

Такое значение данной конфигурационной переменной приведёт к тому, что
таблица текущих результатов будет открываться тегом

`<table width=100%>`

Обратите внимание, что в текущей версии системы `ejudge`, к сожалению,
невозможно использовать знак «кавычки» (") в значении конфигурационной
переменной.
