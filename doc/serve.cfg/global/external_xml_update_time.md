**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"External XML log update interval (sec)"*.

С помощью переменной задается интервал времени в секундах между
генерацией XML-файла во внешнем формате. Файл имеет следующий путь:

<contest_home>`/var/status/dir/external.xml`
