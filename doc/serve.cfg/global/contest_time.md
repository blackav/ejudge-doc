| **Имя переменной**:      | **`contest_time`**                      |     |
| **Содержится в:**        | [`global`](serve.cfg:global.md) |     |
| **Тип содержимого:**     | *integer*                               |     |
| **Может отсутствовать:** | *нет*                                   |     |
| **Может повторяться:**   | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, поле *"Contest time (HH:MM)"*.

**Описание.** Данная конфигурационная переменная задаёт время турнира в
минутах. Время в минутах должно быть целым значением, больше или равным
0. Если время равно 0, продолжительность турнира **не ограничена**. В
этом случае параметры
[`board_fog_time`](serve.cfg:global:board_fog_time.md),
[`board_unfog_time`](serve.cfg:global:board_unfog_time.md) не
используются. Турнир продолжается до тех пор, пока не будет остановлен
администратором.

**Пример.**

`contest_time = 0`

Здесь задаётся неограниченная продолжительность турнира.

`contest_time = 300`

В данном случае продолжительность турнира устанавливается равной 5
часам.
