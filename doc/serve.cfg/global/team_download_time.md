|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`team_download_time`**                |     |
| **Содержится в:**          | [`global`](../global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | `30`                                    |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле *"Team
archive download interval (HH:MM:SS)"*.

**Описание.** Данная конфигурационная переменная задаёт, как часто
участники турнира могут скачивать архив своих решений. Если значение
переменной установлено в 0, загрузка архива своих решений участниками
отключена.

**Пример.**

`team_download_time = 600`
