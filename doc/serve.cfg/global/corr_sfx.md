**Описание.** Данная конфигурационная переменная устанавливает суффикс
имён файлов, в которых хранятся правильные ответы к тестам. Полный путь
к очередному правильному ответу определяется следующим образом:

`path=${prob.corr_dir}/${prob.short_name}/${test_num}${prob.corr_sfx}`

Здесь `prob` — это задача, решаемая тестируемой программой. `test_num` —
это номер теста с тремя десятичными цифрами (включая ведущие незначащие
нули). `${prob.corr_dir}` — это значение конфигурационной переменной
[`corr_dir`](serve.cfg:global:corr_dir.md) задачи. Если описание
задачи не устанавливает эту переменную, используется глобальная
переменная [`corr_dir`](serve.cfg:global:corr_dir.md).
`${prob.short_name}` — это короткое имя задачи (значение
конфигурационной переменной
[`short_name`](serve.cfg:global:short_name.md)) описания задачи.
`${prob.corr_sfx}` — это значение конфигурационной переменной
[`corr_sfx`](serve.cfg:global:corr_sfx.md) описания задачи. Если
в описании задачи эта переменная не установлена, используется глобальная
переменная `corr_sfx`.

**Пример.**

`corr_sfx = ".res"`
