Переменная позволяет задавать начало и конец разных типов файлов
печатных протоколов. Для генерации протокола используется `LaTeX`, таким
образом, в этих файлах должны находиться корректные заголовки и
окончания `LaTeX` файлов.

`user` - это пользовательский протокол, который печатается
непосредственно по окончанию экзамена.
