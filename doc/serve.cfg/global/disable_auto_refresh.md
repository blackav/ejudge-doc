**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Disable auto-refreshing"*.

Если данная глобальная конфигурационная переменная установлена в 1, в
интерфейсе непривилегированного клиента отключается автообновление
страницы, если у пользователя есть непроверенные посылки.

По умолчанию значение данной переменной равно 0, то есть если у
пользователя есть непроверенные посылки, то страница пользователя
автоматически перегружается через 5 секунд. В случае, если на сервере
возникают какие-либо проблемы, препятствующие быстрой проверке
поступающих решений, это приводит к тому, что непривилегированный клиент
практически не может выполнить никаких операций из-за постоянной
перезагрузки веб-страницы.
