Данный конфигурационный параметр позволяет указать плагин, который будет
использоваться для хранении информации о вариантах задач, назначенных
пользователям. По умолчанию используется плагин, указанный в элементе
[`default_variant_plugin`](../../ejudge.xml/default_variant_plugin.md)
глобального конфигурационного файла [ejudge.xml](../../ejudge.xml.md).
Если этот элемент не задан, используется плагин по умолчанию `file`,
который хранит информацию о вариантах в текстовом файле `variant.map`.

Рекомендуется настроить плагин для хранения вариантов в глобальном
конфигурационном файле [ejudge.xml](../../ejudge.xml.md).

<b>Пример.</b>

`variant_plugin = "mysql"`

Поддерживается с версии [3.10.0](../../изменения_в_версии_3.10.0.md).
