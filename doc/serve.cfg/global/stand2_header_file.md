|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`stand2_header_file`**                |     |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *имя файла*                             |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *не установлено*                        |     |
| **Может повторяться:**     | *нет*                                   |     |

**Описание.** Если данная конфигурационная переменная и переменная
[`stand2_file_name`](serve.cfg:global:stand2_file_name.md)
определены, то при генерации вспомогательной таблицы текущего положения
участников в начало генерируемого файла будет помещаться текст,
находящийся в файле-заголовке, полный путь к которому задаётся в этой
конфигурационной переменной. Содержимое файла-заголовка считывается один
раз при старте программы `serve`.

`stand2_header_file = "/home/httpd/ssi/stand2_head.shtml"`
