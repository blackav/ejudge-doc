# `enable_language_import`

Навигация: [Система ejudge](../../система_ejudge.md)/[Использование](../../использование.md)/[Конфигурационные файлы](../../конфигурационные_файлы.md)/[serve.cfg](../../serve.cfg.md)/[Глобальные конфигурационные параметры](../global.md)/[`enable_language_import`](enable_language_import.md)

Данный конфигурационный параметр позволяет задавать список языков
программирования, импортируемых в данный турнир в режиме
[улучшенной поддержки серверов компиляции](../../улучшенная_поддержка_серверов_компиляции.md).

Пример:

```text
enable_language_import
language_import = "enable all"
```

Поддерживается начиная с версии
[3.13.0](../../change/изменения_в_версии_3.13.0.md).
