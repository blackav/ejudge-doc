|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`min_gzip_size`**                     |
| **Содержится в:**          | [`global`](../global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *integer*                               |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | `4096`                                  |
| **Может повторяться:**     | *нет*                                   |
| **Версия `ejudge`:**       | начиная с 2.1.24                        |

**Описание.** Данная конфигурационная переменная задаёт максимальный
размер архивного файла, который не будет сжиматься алгоритмом `gzip`,
если режим сжатия включён (см. глобальную конфигурационную переменную
[`use_gzip`](use_gzip.md)). Значение по
умолчанию является наилучшим для `Linux` на архитектуре `ia32` и
файловой системы `ext2` или `ext3`.
