Данная глобальная конфигурационная переменная позволяет задать число
попыток запуска программы для подтверждения вердитов [TL](TL.md)
или [WT](WT.md). Например, если переменная задана таким образом:

`time_limit_retry_count = 2`

тогда, если при выполнении тестируемой программы на некотором тесте
будет получен вердикт [TL](TL.md) или [WT](WT.md), то
тестируемая программа будет запущена еще один раз. Если и в результате
второго запуска будет получен тот же самый вердикт, он станет
окончательным вердиктом тестирования.

По умолчанию программа запускается на выполнение только один раз на
тест.

Рекомендуется устанавливать данную переменную в значение 2 или 3 в
случае, когда тестирование выполняется на виртуальных машинах и
наблюдается существенная нестабильность во времени работы программы.
