| **Имя переменной**:        | **`serve_sleep_time`**                  |     |
| **Содержится в:**          | [`global`](../global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | равно значению `sleep_time`             |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле
*"\`serve' sleep time (ms)"*.

**Описание.** Данная переменная задаёт интервал времени между опросами
каталогов обмена программы `serve`. Интервал времени задаётся в
миллисекундах. Если конфигурационная переменная `serve_sleep_time` не
установлена, используется значение конфигурационной переменной
[`sleep_time`](sleep_time.md), а если и эта
переменная не установлена, используется значение по умолчанию 1000, что
соответствует одной секунде задержки.
