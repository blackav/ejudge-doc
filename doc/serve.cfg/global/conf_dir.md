|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`conf_dir`**                          |     |
| **Содержится в:**          | [`global`](../global.md) |     |
| **Используется:**          | `serve, compile, run`                   |     |
| **Тип содержимого:**       | *путь к каталогу*                       |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | `conf`                                  |     |
| **Может повторяться:**     | *нет*                                   |     |

**Описание.** Данная конфигурационная переменная позволяет задать
каталог, в котором находятся конфигурационные файлы, тесты, проверяющие
программы, ответы к тестам. Если эта переменная не задана,
конфигурационным каталогом является подкаталог `conf` каталога,
заданного в переменной `root_dir`. Если конфигурационная переменная
`conf_dir` задана, полный путь к каталогу конфигурационных файлов
образуется конкатенацией пути, указанного в переменной `root_dir` и
пути, указанного в переменной `conf_dir`.
