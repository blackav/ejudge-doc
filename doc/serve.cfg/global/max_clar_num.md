| **Имя переменной**:        | **`max_clar_num`**                      |     |
| **Содержится в:**          | [`global`](../global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | 50                                      |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's quotas"*, поле
*"Maximum number of clarification requests"*.

**Описание.** Данная конфигурационная переменная устанавливает
максимальное количество сообщений от одного участника. Если максимальное
количество сообщений превышено, очередное сообщение будет отвергнуто, а
участнику будет выдано сообщение об ошибке. Значение по умолчанию — 50,
чего должно быть достаточно для разумного использования.
