|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`a2ps_path`**                         |
| **Содержится в:**          | [`global`](../global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *путь к программе*                      |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | `/usr/bin/a2ps`                         |
| **Может повторяться:**     | *нет*                                   |
| **Версия `ejudge`:**       | начиная с 2.1.26.                       |

**Описание.** Данная конфигурационная переменная позволяет задать путь к
программе `a2ps`, которая используется для преобразования текста
программы в формат PostScript для печати на принтере.

**Пример.**

`a2ps_path = "/usr/local/bin/a2ps"`
