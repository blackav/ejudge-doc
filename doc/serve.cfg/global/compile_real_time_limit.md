|                            |                                         |     |
|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`compile_real_time_limit`**           |     |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |     |
| **Используется:**          | `compile`                               |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | `30`                                    |     |
| **Может повторяться:**     | *нет*                                   |     |

**Описание.** Данная конфигурационная переменная задаёт ограничение
астрономического времени на компиляцию программы участника. В случае,
если максимальное время компиляции превышено, проверка программы
участника завершается со статусом “Check failed” («Проверка не
удалась»). Значение 0 означает отсутствие ограничения времени.

**Пример.**

`compile_real_time_limit = 60`
