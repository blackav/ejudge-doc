**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле
*"Memoize user results"*.

Если данная переменная установлена в 1, в фильтре посылок становится
возможным использование конструкции `total_score`, возвращающей целое
значение - суммарный балл участника.
