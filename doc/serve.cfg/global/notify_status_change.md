(Global settings/Contestant's capabilities/Enable e-mail status change
notifications).

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Enable e-mail status change notifications"*.

Если данная переменная установлена, то при изменении статуса посылки
пользователя ему по электронной почте направляется уведомительное
письмо.
