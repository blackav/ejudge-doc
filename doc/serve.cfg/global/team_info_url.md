| **Имя переменной**:        | **`team_info_url`**                     |     |
| **Содержится в:**          | [`global`](../global.md) |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *шаблон URL*                            |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *не установлено*                        |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"URL to view detailed contestant information"*.

**Описание.** Данная конфигурационная переменная задаёт шаблон URL, по
которому находится более подробная информация об участнике. Этот шаблон
раскрывается в момент генерации таблицы положения участников в турнире и
формирует гиперссылку от имени участника и его идентификатора к
отдельной странице с подробной информацией о команде. Подробная
информация о поддерживаемых форматных преобразованиях дана в разделе
[Форматные подстановки](../../форматные_подстановки.md).

**Пример**.

`team_info_url = "http://www.contest.ru/users/%Mi"`
