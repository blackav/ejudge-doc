|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`stand_v_row_attr`**                  |
| **Содержится в:**          | [`global`](../global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *string*                                |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | "" (пустая строка)                      |
| **Может повторяться:**     | *нет*                                   |

**Описание.** Данная конфигурационная переменная позволяет установить
дополнительные атрибуты строк таблицы результатов виртуального турнира,
соответствующих виртуальным участникам виртуального турнира. Значение
данной переменной добавляется к элементу <tt>

<tr>

</tt> этой строки.

**Пример.**

`stand_v_row_attr = " bgcolor=#eeffff"`

Такое значение данной конфигурационной переменной приведёт к тому, что
строки таблицы результатов будет открываться тегом <tt>

<tr bgcolor=#eeffff>

</tt>.

**Обратите внимание**, что в текущей версии системы `ejudge`, к
сожалению, невозможно использовать знак «кавычки» (") в значении
конфигурационной переменной.
