Позволяет задавать атрибуты столбцов таблицы ссылок на страницы таблицы
текущих результатов, если включено разбиение таблицы текущих результатов
на страницы.

Эта переменная должна быть определена 2 раза, формируя массив атрибутов
из 2 элементов. Массив используется следующим образом:

- `stand_page_col_attr[0]` - атрибут первого столбца таблицы (названия
  строк),
- `stand_page_col_attr[1]` - атрибут всех остальных столбцов таблицы.

**Пример** .

`stand_page_col_attr = ""`  
`stand_page_col_attr = " width=\"60\""`
