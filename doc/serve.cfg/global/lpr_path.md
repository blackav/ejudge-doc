|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`lpr_path`**                          |
| **Содержится в:**          | [`global`](../global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *путь к программе*                      |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | `/usr/bin/lpr`                          |
| **Может повторяться:**     | *нет*                                   |
| **Версия `ejudge`:**       | начиная с 2.1.26.                       |

**Описание.** Данная конфигурационная переменная позволяет задать путь к
программе `lpr`, которая используется для постановки PostScript-файла в
очередь печати на принтер.

**Пример.**

`lpr_path = "/usr/local/bin/lpr"`
