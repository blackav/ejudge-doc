Данный параметр позволяет задать максимальный размер пользовательских
данных при [запуске на пользовательских
данных](../../запуск_на_пользовательских_данных.md) одного
пользователя в одном турнире.

Значение параметра по умолчанию равно 1024.

<b>Пример:</b>

`max_input_size = 512`

Поддерживается начиная с версии
[3.10.0](../../изменения_в_версии_3.10.0.md).
