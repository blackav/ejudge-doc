Команда unload выгружает структуры данных, относящиеся к турниру, из
памяти сервера. Поскольку структуры данных турнира автоматически
подгружаются в память при запросе к турниру, данная команда работает как
кнопка "Reload contest settings" административного web-интерфейса, то
есть позволяет перечитать конфигурационные файлы турнира. Для выполнения
команды необходимы полномочия администратора турнира.

Использование:

`ejudge-contests-cmd CONTEST-ID unload [OPTIONS] SESSION-FILE`

Поддерживаются все [стандартные опции
команд](ejudge-contests-cmd:_COMMAND-OPTIONS.md). В частности,
если указана опция `--session`, то аргумент `SESSION-FILE` — это не
файл, в котором записан сессионный ключ, а сам сессионный ключ.

Начиная с версии [2.3.20](Изменения_в_версии_2.3.20.md)
поддерживается упрощенная команда
[unload-2](ejudge-contests-cmd:_unload-2.md).
