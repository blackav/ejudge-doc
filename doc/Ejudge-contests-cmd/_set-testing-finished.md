Команда set-testing-finished устанавливает флаг окончания тестирования.
Данная команда доступна только администратору турнира.

Флаг окончания тестирования турнира действует только в турнирах по
системе OLYMPIAD. Если этот флаг не установлен, то считается, что
окончательное тестирование решений еще не окончено. Поэтому на личных
страницах участников турнира будет отображаться только информация о
приёме их решений на проверку, даже если соответствующие решения уже
были полностью протестированы. Поэтому установка флага окончания турнира
открывает участникам турнира доступ к полным протоколам проверки их
решений.

Использование:

`ejudge-contests-cmd CONTEST-ID set-testing-finished [OPTIONS] SESSION-FILE`

Поддерживаются все [стандартные опции
команд](ejudge-contests-cmd:_COMMAND-OPTIONS.md). В частности,
если указана опция `--session`, то аргумент `SESSION-FILE` — это не
файл, в котором записан сессионный ключ, а сам сессионный ключ.
