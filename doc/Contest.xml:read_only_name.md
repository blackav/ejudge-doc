| **Имя атрибута**:          | **`read_only_name`**                        |     |
| **Содержится в:**          | [`contest`](contest.xml:contest "wikilink") |     |
| **Тип значения:**          | *boolean*                                   |     |
| **Может отсутствовать:**   | *да*                                        |     |
| **Значение по умолчанию:** | false                                       |     |

**Описание.** Если этот атрибут установлен в *true* или *yes*, в данном
турнире пользователи не могут редактировать поле

    name

. Все остальные поля, например, аватар, остаются доступными для
редактирования. Опция полезна, когда имена пользователей загружаются
администратором из внешнего источника, и нежелательно, чтобы
пользователи могли его изменять.

Поддерживается с версии [3.9.0](изменения_в_версии_3.9.0 "wikilink").
