Если для некоторого языка программирования данная конфигурационная
переменная установлена в положительное значение, при тестировании
программ, написанных на данном языке, будет выполняться преобразование
концов строк из Unix-формата (\n) в Dos-формат (\r\n). Преобразование
подавляется, если у задачи установлен флаг бинарного режима.

Пример:

`[language]`  
`short_name = "pasabc-linux"`  
`is_dos`
