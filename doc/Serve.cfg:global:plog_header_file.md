|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`plog_header_file`**                  |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *имя файла*                             |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *не установлено*                        |     |
| **Может повторяться:**     | *нет*                                   |     |

**Описание.** Если данная конфигурационная переменная и переменная
[`plog_file_name`](serve.cfg:global:plog_file_name "wikilink")
определены, то при генерации журнала посылок турнира в начало
генерируемого файла будет помещаться текст, находящийся в
файле-заголовке, полный путь к которому задаётся в этой конфигурационной
переменной. Содержимое файла-заголовка считывается один раз при старте
программы `serve`.

`plog_header_file = "/home/httpd/ssi/plog_head.shtml"`
