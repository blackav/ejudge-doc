| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |

| **Тип содержимого:**       | *boolean*                               |

| **Версия `ejudge`:**       | начиная с 2.1.24                        |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's capabilities"*,
поле *"Contestant may view compilation errors"*.

**Описание.** Если данная конфигурационная переменная установлена в
значение *true*, а глобальная конфигурационная переменная
[`team_enable_rep_view`](serve.cfg:global:team_enable_rep_view "wikilink")
установлена в *false*, участникам предоставляется доступ к протоколу
компиляции их посылки в случае, если компиляция завершилась из-за
ошибки. Для всех статусов посылки, кроме ошибки компиляции, протокол
тестирования участникам недоступен. Если глобальная конфигурационная
переменная `team_enable_rep_view` установлена в *true*, значение данной
конфигурационной переменной не используется.

Для каждой задачи режим показа ошибок компиляции может быть
переопределён с помощью конфигурационной переменной
[`team_enable_ce_view`](serve.cfg:global:team_enable_ce_view "wikilink")
секции описания задачи.
