|----------------------------|-----------------------------------------|-----|
| **Имя переменной**:        | **`enable_continue`**                   |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *boolean*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | *false*                                 |     |
| **Может повторяться:**     | *нет*                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле
*"Enable contest continuation"*.

**Описание.** Если данная конфигурационная переменная установлена в
*true*, и продолжительность турнира не ограничена (см. конфигурационную
переменную [`contest_time`](serve.cfg:global:contest_time "wikilink")),
администратор турнира получает возможность продолжить турнир после того,
так он был остановлен. Во всех других случаях остановленный или
завершившийся турнир не может быть продолжен.

**Пример.**

`enable_continue`
