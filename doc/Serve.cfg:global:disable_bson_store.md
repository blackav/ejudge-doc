Если данный глобальный конфигурационный параметр установлен в
положительное значение, протоколы запуска будут храниться в формате XML,
даже если при компиляции в системе была обнаружена библиотека libbson.

В противном случае, если при компиляции ejudge была включена поддержка
BSON, для хранения протоколов тестирования будет использоваться формат
BSON.

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0 "wikilink").
