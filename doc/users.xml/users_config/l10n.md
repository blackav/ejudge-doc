|----------------------------|-----------------------------------------------------|
| **Имя атрибута**:          | **`l10n`**                                          |
| **Содержится в:**          | [`users_config`](../users_config.md) |
|                            |                                                     |
| **Тип значения:**          | *boolean*                                           |
| **Значение по умолчанию:** | устанавливается [`configure`](configure.md) |
| **Может отсутствовать:**   | *да*                                                |
| **Примечание:**            | явное использование не рекомендуется                |

**Описание.** Данный атрибут включает поддержку локализации интерфейса.
Если его значение установлено в *true*, CGI-программа
[users](../../users.md) считывает значение параметра `locale_id` и
переключает язык интерфейса в зависимости от значения параметра.
Подробнее вопросы локализации системы рассматриваются в разделе
[Локализация системы](../../локализация_системы.md).

Логическое значение *true* задаётся с помощью строк `1, yes` или `true`,
а логическое значение *false* — с помощью строк `0, no, false`.

Явно устанавливать данный атрибут не рекомендуется. Для запрета
локализации системы предпочтительнее использовать опцию `--disable-nls`
скрипта [configure](configure.md) при компиляции системы
`ejudge`.
