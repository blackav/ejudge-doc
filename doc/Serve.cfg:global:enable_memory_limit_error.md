**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, поле *"Enable support for MemoryLimit
error"*.

Если эта переменная установлена, то при запуске программ на тестирование
в данном турнире включается поддержка обнаружения ошибки превышения
максимально допустимого объёма памяти.
