**Описание.** Значение данного элемента содержит полный путь к рабочему
каталогу системы `ejudge`. В рабочем каталоге располагаются файлы
журналов программ системы, а также рабочие каталоги программ системы.

Рекомендуемое значение данного пути устанавливается автоматически на
этапе первоначальной настройки системы программой
[`ejudge-setup`](ejudge-setup.md) на основе значения, указанного
в опции `--enable-contests-home-dir`. После этого изменять значение
этого элемента не рекомендуется.
