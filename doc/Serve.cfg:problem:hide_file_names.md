Если данная конфигурационная переменная установлена в положительное
значение, то при выводе условия задач на странице сдачи решения
участника турнира не отображаются имена входного и выходного файла, даже
если они заданы в конфигурационном файле турнира.

Конфигурационная переменная поддерживается, начиная с версии
[3.1.0](изменения_в_версии_3.1.0.md).
