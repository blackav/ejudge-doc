| **Имя элемента**:        | **`cap`**                                                                                |     |
| **Содержится в:**        | [`caps`](ejudge.xml:caps.md)                                                     |     |
| **Может содержать:**     | *нет*                                                                                    |     |
| **Атрибуты:**            | [`login`](ejudge.xml:caps.md)                                                    |     |
| **Тип содержимого:**     | список битов полномочий                                                                  |     |
| **Может отсутствовать:** | *да*                                                                                     |     |
| **Может повторяться:**   | *да*                                                                                     |     |
| **Используется в:**      | [`userlist-server`](userlist-server.md). [`super-serve`](super-serve.md) |     |

**Описание.** Данный элемент позволяет задавать полномочия на выполнение
операций над базой данных пользователей.

Атрибут `login` задаёт регистрационное имя пользователя, для которого
устанавливаются полномочия.

**Пример.**

<cap login="dima">`MAP_CONTEST`</cap>
