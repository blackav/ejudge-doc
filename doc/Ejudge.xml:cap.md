**Описание.** Данный элемент позволяет задавать полномочия на выполнение
операций над базой данных пользователей.

Атрибут `login` задаёт регистрационное имя пользователя, для которого
устанавливаются полномочия.

**Пример.**

<cap login="dima">`MAP_CONTEST`</cap>
