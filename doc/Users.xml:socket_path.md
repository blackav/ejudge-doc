**Описание.** Данный элемент задаёт путь к UNIX-сокету, который
используется для связи с сервером информации о пользователях
[userlist-server](userlist-server "wikilink"). Когда программа
[register](register "wikilink") стартует, UNIX-сокет должен существовать
и быть доступным для пользователя, от имени которого запускается
программа. Как правило, это пользователь, от имени которого работает
WEB- сервер.

Правильное значение по умолчанию пути к сокету автоматически
устанавливается про- граммой [configure](configure "wikilink") при
компиляции системы, если задана опция `--enable-socket-path`. Поэтому
явное задание данной опции не рекомендуется.
