языков](serve.cfg:language "wikilink")/[`run_max_file_size`](Serve.cfg:language:run_max_file_size "wikilink")

Если данный конфигурационный параметр языка программирования установлен
в положительное значение, при выполнении программ на данном языке будет
использоваться указанное ограничение на максимальный размер файла, а не
ограничение, указанное в параметре
[`max_file_size`](Serve.cfg:problem:max_file_size "wikilink").

Данный конфигурационный параметр может быть полезным при запуске
программ на языках .Net. По непонятной причине программы на
[C#](Lang:dotnet-cs "wikilink") не запускаются, если это ограничение
меньше `4096G`.

Поддерживается начиная с версии
[3.13.0](изменения_в_версии_3.13.0 "wikilink").
