**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings files and URLs"*, поле
*"Number of users on standings page"*.

Если эта переменная установлена в положительное значение, таблица
результатов турнира будет разбиваться на страницы по заданному числу
пользователей на страницу. Пока разбиение на страницы работает только
для турниров по системе *KIROV*.
