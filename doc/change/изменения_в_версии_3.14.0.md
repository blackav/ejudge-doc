После инсталляции новой версии ejudge обязательно выполните команды
`ejudge-suid-setup` и `ejudge-upgrade-web` из-под пользователя root и
команду `ejudge-configure-compilers` из-под пользователя ejudge.

## Главные новые возможности и улучшения

Новый компонент [ej-agent-server](../tool/ej-agent-server.md) для организации распределённой
компиляции и тестирования. С помощью него поддерживается взаимодействие агентов
компиляции и тестирования и сервера турниров по протоколу websocket. Использование
протокола websocket упрощает развёртывание агентов, кроме того при использовании
протокола websocket реализованы строгие ограничения на файлы, доступные для скачивания
агентами.

Поддерживается формат Markdown для условий задач.

Реализован [API для конфигурирования турниров](../api/API.md).

Новая утилита [ejudge-problem](../tool/ejudge-problem.md) для подготовки задач.
