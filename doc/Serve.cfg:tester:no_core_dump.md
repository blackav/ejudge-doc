|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`no_core_dump`**                      |
| **Содержится в:**          | [`tester`](serve.cfg:tester "wikilink") |
| **Используется:**          | `run`                                   |
| **Тип содержимого:**       | *boolean*                               |
| **Может отсутствовать:**   | *да*                                    |
| **Наследуется:**           | *да*                                    |
| **Значение по умолчанию:** | *false*                                 |
| **Может повторяться:**     | *нет*                                   |

**Описание.** Если данная конфигурационная переменная установлена в
значение *true*, при запуске тестируемой программы запрещается генерация
дампа памяти (core dump) при аварийном завершении тестируемой программы.
