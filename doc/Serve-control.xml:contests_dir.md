**Описание.** Данный элемент задаёт путь к каталогу, в котором хранятся
описания турниров [`contest.xml`](contest.xml "wikilink"). Каждое
описание турнира находится в отдельном файле с именем, равным
шестизначному номеру турнира, например, `000001.xml`.

Правильное значение по умолчанию пути к каталогу описания турниров
автоматически устанавливается программой
[configure](configure "wikilink") при компиляции системы, используя
значения опций `--enable-conf-dir` или `--enable-contests-home-dir`.
Поэтому явное задание данной опции не рекомендуется.
