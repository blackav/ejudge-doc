| **Имя переменной**:        | **`src_sfx`**                               |
| **Содержится в:**          | [`language`](serve.cfg:language "wikilink") |
| **Используется:**          | `serve`                                     |
| **Тип содержимого:**       | *string*                                    |
| **Может отсутствовать:**   | *да*                                        |
| **Значение по умолчанию:** | "" (пустая строка)                          |
| **Может повторяться:**     | *нет*                                       |

**Описание.** Данная конфигурационная переменная устанавливает суффикс,
который должны иметь файлы, подаваемые на компиляцию данному компилятору
языка программирования. Например, компилятор `GCC` требует, чтобы файл
программы на языке `Си` имел суффикс `.c.` а файл программы на языке
`Си++` имел суффикс `.cpp`, `.cc` или `.C`.

**Пример.**

`src_sfx = ".c"`
