**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Files and directories"*, поле
*"Contest stop script"*.

Данная конфигурационная переменная позволяет указывать команду, которая
будет выполнена в момент завершения турнира. Если значением переменной
является относительный путь, он отсчитывается относительно каталога
конфигурационных файлов `conf` турнира.

Если турнир был остановлен, затем продлен, затем снова остановлен,
команда завершения турнира будет выполнена столько раз, сколько турнир
был остановлен.
