**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"extra HTML attributes for
CGI-programs"*, поле *"Format specification for users table"*.

**Описание.** Данный элемент позволяет управлять видом таблицы
пользователей, зарегистрированных на турнир. Элемент используется, когда
таблица зарегистрированных пользователей выводится на локальном языке
(`locale_id` не равно 0).

Элемент позволяет изменять информацию, которая выводится в колонках
таблицы со второй по предпоследнюю. Первая колонка таблицы всегда
называется `NN` (порядковый номер в списке), а последняя - `Status` (или
`Статус`) — статус регистрации участника. Значение элемента — это
шаблоны [форматной подстановки](Форматные_подстановки "wikilink") для
каждого столбца со второго по предпоследний. Шаблоны столбцов
разделяются знаком `|` (вертикальная черта). Например, спецификация

<users_table_format>`%Ui|%Un|%Ut|%Uf`</users_table_format>` `

означает, что таблица будет состоять из 6 столбцов: порядкового номера,
идентификатора пользователя, имени пользователя, краткой формы записи
учебного заведения, краткой формы записи факультета, статуса
регистрации. Из всех спецификаций форматной подстановки допускаются
только спецификации подстановки информации о пользователе `(%U*)`.

Если значение данного элемента задано, обязательно должно быть задано
значение элемента
[`users_table_legend`](contest.xml:users_table_legend "wikilink"), при
этом оба элемента должны определять равное число стобцов.
