|                            |                                            |
|----------------------------|--------------------------------------------|
| **Имя атрибута**:          | **`allow`**                                |
| **Содержится в:**          | [`access`](register.xml:access "wikilink") |
|                            |                                            |
| **Тип значения:**          | *boolean*                                  |
| **Значение по умолчанию:** | *false*                                    |
| **Может отсутствовать:**   | *да*                                       |

**Описание.** Данный атрибут задаёт права доступа в случае, если
IP-адрес клиента удовлетворяет спецификации IP-адреса в данном элементе.
Если значение атрибута равно *true*, такой IP-адрес считаются
допустимым, а если значение атрибут установлено в *false*, доступ с
этого IP-адреса запрещается. См. подробное описание ограничений
IP-адреса в разделе [Ограничение доступа по
IP-адресам](Ограничение_доступа_по_IP-адресам "wikilink").
