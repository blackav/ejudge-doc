Динамические задачи — это задачи, условия и тесты которых генерируются
"на лету". Динамические задачи могут генерироваться уникально и
индивидуально для каждого участника турнира, либо могут генерироваться в
большом количестве вариантов. Этого сложно достичь с помощью [вариантной
задачи](serve.cfg:problem:variant_num "wikilink").
