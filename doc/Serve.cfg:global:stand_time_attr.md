|                            |                                         |
|----------------------------|-----------------------------------------|
| **Имя переменной**:        | **`stand_time_attr`**                   |
| **Содержится в:**          | [`global`](serve.cfg:global.md) |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *string*                                |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | "" (пустая строка)                      |
| **Может повторяться:**     | *нет*                                   |

**Описание.** Данная конфигурационная переменная позволяет определить
атрибуты, с которыми выводится время сдачи задачи в таблице результатов
турнира по системе *ACM*. Значение данной переменной добавляется к
элементу `<div>` элементов таблицы. Значение данной переменной и
элементы `div` не разделяются дополнительными пробелами, поэтому
значение данной переменной должно начинаться с пробельного символа.

**Пример.**

`stand_time_attr = " class=st_time"`

Такое значение данной конфигурационной переменной приведёт к тому, что
время сдачи задачи будет открываться тегом `<div class=st_time>`.

**Обратите внимание**, что в текущей версии системы `ejudge`, к
сожалению, невозможно использовать знак «кавычки» (") в значении
конфигурационной переменной.
