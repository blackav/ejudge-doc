**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Basic contest
identification"*, поле *"Main URL"*.

Этот элемент позволяет задавать URL для главной страницы турнира и
используется в письме с регистрационными данными, отправляемом при
регистрации нового пользователя на турнир, например

`<main_url>http://acm.univ.ru/contests/acm2006</main_url>`
