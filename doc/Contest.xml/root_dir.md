**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Advanced path settings"*,
поле *"The contest root directory"*.

**Описание.** Данный элемент позволяет задавать домашний каталог
турнира. В подкаталогах домашнем каталоге турнира размещается
конфигурационный файл турнира [`serve.cfg`](serve.cfg.md),
тесты, проверяющие программы, базы посылок и т. д.

Этот элемент должен быть установлен для того, чтобы программа
[`super-serve`](super-serve.md) могла управлять этим турниром.
Если значение элемента не установлено или не является абсолютным путём,
корректная работа системы `ejudge` не гарантируется.

При создании нового турнира с помощью программы
[`serve-control`](serve-control.md) домашний каталог для нового
турнира устанавливается и создаётся автоматически, основываясь на
значении опции `--enable-contests-home-dir` программы
[`configure`](configure.md).
