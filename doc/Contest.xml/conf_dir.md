**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Advanced path settings"*,
поле *"The contest configuration directory"*.

**Описание.** Данный элемент позволяет задавать каталог конфигурационных
файлов турнира. Если значение элемента не задано, используется значение
по умолчанию `conf`. Если значение элемента (включая значение по
умолчанию) является относительным путем, абсолютный путь вычисляется
относительно домашнего каталога турнира, задаваемого с помощью элемента
[`root_dir`](contest.xml:root_dir.md). Рекомендуется всегда
сохранять значение по умолчанию.

**Пример.**

<root_dir>`/home/judges/000111`</root_dir>
