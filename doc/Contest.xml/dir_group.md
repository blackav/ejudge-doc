**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"General settings (contest.xml)"*, блок *"Advanced path settings"*,
поле *"The directory group"*.

Данный элемент позволяет настраивать права доступа и идентификатор
группы для создаваемых системой файлов и каталогов.
