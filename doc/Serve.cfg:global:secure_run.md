**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, поле *"Run programs securely"*.

Если данная глобальная конфигурационная переменная установлена в
положительное значение, то программы на тестирование запускаются в
безопасном режиме. Особенности установки безопасного режима зависят от
языка программирования.

Безопасный режим, установленный для всего турнира в целом, может быть
отменен для отдельных задач с помощью конфигурационной переменной
[`disable_security`](serve.cfg:problem:disable_security "wikilink")
раздела описания задачи, а для отдельных языковых процессоров с помощью
конфигурационной переменной
[`disable_security`](serve.cfg:language:disable_security "wikilink")
раздела описания языкового процессора.
