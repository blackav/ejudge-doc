**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Advanced settings"*, поле
*"Disable user database loading"*.

Если данная конфигурационная переменная установлена в положительное
значение, для данного турнира не будет подгружаться база пользователей,
то есть во всех таблицах, в которых отображается логин или имя
участника, будет отображаться только его идентификатор пользователя.

Тем не менее, все права пользователей на вход в турнир и на выполнения
привилегированных действий будут проверяться по-прежнему. На личной
странице участника будет по-прежнему отображаться его логин и имя.

Данная конфигурационная переменная предназначена для уменьшения
потребления памяти сервером турниров в случае, если, например, таблица
текущих результатов турнира отображается не средствами ejudge, а другим
образом.

Данная конфигурационная переменная поддерживается, начиная с версии
[2.3.23](изменения_в_версии_2.3.23.md).
