Пробельный символ — один из следующих символов:

- \f (\x0c, код 12) — form feed
- \n (\x0a, код 10) — newline
- \r (\x0d, код 13) — carriage return
- \t (\x09, код 9) — horizontal tab
- \v (\x0b, код 11) — vertical tab
- ' ' (\x20, код 32) — space

В зависимости от текущей локали множество пробельных символов может
варьироваться. Указанные символы даны для стандартной локали (C или
POSIX) языка C.

В системе ejudge программы на языке [java](java.md) запускаются
в локали C, чтобы избежать проблем с чтением вещественных чисел.
Программы на остальных языках запускаются в текущей локали.
