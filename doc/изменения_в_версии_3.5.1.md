### Общие исправления

Устранены несколько случаев утечки памяти.

### [ej-compile](ej-compile.md)

Добавлена поддержка транслятора `gas` - GNU Assembler.

В скриптах компиляции для языков `gcc`, `g++` и некоторых других удалена
опция `-static`.

Поддерживается [потестовая компиляция](потестовая_компиляция.md)
решений, то есть для каждого теста генерируется свой исполняемый файл. В
режиме потестовой компиляции вместо одного исполняемого файла на выходе
программы [ej-compile](ej-compile.md) получается архив в формате
.zip с скомпилированной программой для каждого теста.

В целом исходный текст программы подвергнут переработке для улучшения
общей структуры.

### [ej-contests](ej-contests.md)

Реализована поддержка [потестовой
компиляции](потестовая_компиляция.md) решений.

Исправлена ошибка с некорректным показом числа непрочитанных вопросов
при некоторых условиях.

### [ej-super-run](ej-super-run.md)

При запуске тестируемых программ на выполнение переменные окружения
`LANG` и `LC_CTYPE` устанавливаются в значение `en_US.UTF-8`. Это
позволяет программам на языке python корректно обрабатываться ввод и
вывод в кодировке UTF-8.
