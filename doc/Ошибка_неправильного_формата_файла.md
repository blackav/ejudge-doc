Ошибка [неправильного формата
файла](Неправильный_формат_файла "wikilink") возникает при чтении данных
из одного из входных файлов проверяющей программы, когда формат данных
не соответствует формату, ожидаемому проверяющей программой.

Если эта ошибка возникла при чтении из файла входных данных или файла
правильного ответа, диагностируется [внутренняя ошибка
проверки](внутренняя_ошибка_проверки "wikilink").

Если эта ошибка возникла при чтении из файла результата работы
тестируемой программы, диагностируется [ошибка неправильного формата
результата](ошибка_неправильного_формата_результата "wikilink").

См. также: [`fatal_read`](libchecker:fatal_read "wikilink").
