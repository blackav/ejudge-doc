Английское название: [Style Violation](style_violation.md), [SV](SV.md)

Данная ошибка может диагностироваться [проверяющей программой для стиля
оформления исходного кода](../style_checkers.md), которая работает
на этапе компиляции решения.

Кроме того, данная ошибка может устанавливаться вручную администратором
системы после просмотра исходного кода программы, сданной на проверку.
