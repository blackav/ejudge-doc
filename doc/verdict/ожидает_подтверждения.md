Английское название: [Pending review](Pending_review.md),
[PR](PR.md)

Данный статус проставляется для посылок по задачам, для которых
установлен флаг
[`use_ac_not_ok`](../serve.cfg/problem/use_ac_not_ok.md), если
посылка проходит все тесты. Посылка с таким статусом предназначена для
просмотра преподавателем или администратором турнира, который должен для
нее установить окончательный статус Ok, Rejected, Ignored, Disqualified
и т. п.
