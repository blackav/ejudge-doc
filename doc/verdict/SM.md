Аббревиатура: SM

Выражение фильтра посылок: `status == SM`

Статус: Summoned for defence, Вызван на защиту

Данный статус может быть установлен преподавателем, проверяющим работы.
Статус может использоваться для вызова ученика или студена на личную
защиту сданного решения.
