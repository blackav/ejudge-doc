# Security violation

Навигация: [Система ejudge](../система_ejudge.md)/[Использование](../использование.md)/[Вердикты тестирования](../вердикты_тестирования.md)/[Ошибка нарушения ограничений безопасности](ошибка_нарушения_ограничений_безопасности.md)

Английское название: [Security Violation](security_violation.md), [SE](SE.md)

Данная ошибка может диагностироваться при запуске тестируемой программы,
если она пытается выполнить недопустимый системный вызов, например,
создать сокет или создать процесс.

Диагностика данной ошибки может быть включена в режиме контейнерного исполнения
с помощью конфигурационного параметра
[`detect_violations`](../serve.cfg/global/detect_violations.md).

**Устарело** Диагностика данной ошибки включена только в режиме безопасного
выполнения программ
[`secure_run`](../serve.cfg/global/secure_run.md) с включенной
опцией
[`detect_violations`](../serve.cfg/global/detect_violations.md).
