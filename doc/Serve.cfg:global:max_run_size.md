| **Имя переменной**:        | **`max_run_size`**                      |     |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |     |
| **Используется:**          | `serve`                                 |     |
| **Тип содержимого:**       | *integer*                               |     |
| **Может отсутствовать:**   | *да*                                    |     |
| **Значение по умолчанию:** | 65535                                   |     |

**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Contestant's quotas"*, поле
*"Maximum size of one submitted program"*.

**Описание.** Данная конфигурационная переменная устанавливает
максимальный размер программы, принимаемой на проверку. Если размер
программы превышает лимит, программа к проверке не принимается, а
участнику выводится сообщение об ошибке. При увеличении максимального
размера программы учтите, что действует ограничение размера данных,
принимаемых от клиента, равное 128 килобайт. Чтобы изменить это
ограничение исправьте значение константы `MAX_VALUE_SIZE` в исходном
файле `cgi.c` и перекомпилируйте систему `ejudge`.

**Пример.** Данный пример устанавливает максимальный размер программы
равным 100 килобайт.

`max_run_size = 102400`
