|----------------------------|----------------------------------------------------------------------------------------------------------|
| **Имя параметра:**         | **`l10n_dir`**                                                                                           |
| **Тип содержимого:**       | путь к каталогу                                                                                          |
| **Может отсутствовать:**   | *да*                                                                                                     |
| **Значение по умолчанию:** | устанавливается программой [configure](configure "wikilink")                                             |
| **Может повторяться:**     | *нет*                                                                                                    |
| **Описание:**              | См. раздел [Включение поддержки языкового окружения](Включение_поддержки_языкового_окружения "wikilink") |
| **Замечание:**             | В текущей версии программы `master` данный параметр не имеет никакого эффекта.                           |

Если параметр [`enable_l10n`](master.cfg:enable_l10n "wikilink")
установлен в *true*, а параметр `l10n_dir` не задан, локализация
сообщений принудительно отключается.

**Пример.**

`enable_l10n`  
`l10n_dir = "/usr/share/locale"`
