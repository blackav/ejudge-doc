**Описание.** Данный элемент задаёт путь к каталогу, в котором хранятся
локализованные сообщения. Для локализации сообщений используется
`GNU gettext`. См. раздел [Локализация
системы](../локализация_системы.md) для информации о локализации
системы.

Корректный путь устанавливается автоматически при компиляции системы с
помощью скрипта [configure](configure.md), поэтому явное
использование этого элемента не рекомендуется.
