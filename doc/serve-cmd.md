Программа `serve-cmd` предоставляет интерфейс командной строки для
выполнения некоторых административных и пользовательских операций в
активном турнире. В настоящий момент реализовано подмножество операций,
доступных с помощью CGI-программ [`master`](master.md) и
[`team`](team.md).
