role - строка, задающая роль участника команды. Может принимать одно из
следующих значений.

`"CONTESTANT"   участник команды, либо собственно участник в случае личных турниров`  
`"RESERVE"      запасной участник команды`  
`"COACH"        тренер команды`  
`"ADVISOR"      руководитель команды`  
`"GUEST"        гость`
