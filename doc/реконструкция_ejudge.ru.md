## Список работ на сайте www.ejudge.ru

### Страница download

- Вписать ее в общий дизайн сайта
- Вверху страницы показывать последние версии программ по всем
  категориям (ejudge, reuse, ejudge-doc, kernel-patch)
- При выборе какой-нибудь категории показывать страницу с архивами
  последней версии для всех поддерживаемых архитектур
- Для каждой категории сделать ссылку "Все версии"
- При показе всех версий отображаются номера версий с датой релиза,
  можно каждую версию раскрыть, тогда будут показаны архивы для всех
  архитектур.
- Список архивов берется прямо из каталога архивов. Дата релиза
  опреледяется по дате выкладывания файла, номер версии и архитектура
  извлекаются из имени файла.
