Исправление предупреждений компилятора gcc версии 8.

Ослаблена проверка на корректные логины. В частности, допускаются логины
с пробелами в середине.

Уничтожаются все файлы, которые передавались от компонента компиляции в
качестве результата. В предыдущих версиях неудаленные файлы постепенно
накапливались в каталогах обмена.

Компилятор java версии 10 поддерживается.

Компилятор Microsoft Visual C# (csc) поддерживается.

Обновления для взаимодействия с текущей версией системы Polygon.

Поддерживаются последние версии ядра Linux. Из-за исправления в ядре
(https://lore.kernel.org/patchwork/patch/855414/) лимит стека
сбрасывался в лимит по умолчанию (8 мб) при запуске программы на
тестирование в режиме
[`enable_suid_run`](serve.cfg:problem:enable_suid_run "wikilink").
