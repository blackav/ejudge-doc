В системе ejudge поддерживается выполнение большей части действий из
командной строки, что дает широкие возможности для автоматизации
управления системой. Кроме того, обратите внимание, что для доступа к
базе пользователей предоставляется [модуль
расширения](python/_Работа_с_базой_пользователей.md) для языка
Питон.

- [ejudge-contests-cmd](ejudge-contests-cmd.md) - доступ к
  серверу турниров из командной строки
- [ejudge-execute](ejudge-execute.md) - контролируемый запуск
  программ
