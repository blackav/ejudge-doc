Команда dump-problems выводит на стандартный поток вывода список всех
задач турнира. Для выполнения этой команды необходимы полномочия
администратора турнира.

Использование:

`ejudge-contests-cmd CONTEST-ID dump-problems [OPTIONS] SESSION-FILE`

Поддерживаются все [стандартные опции
команд](ejudge-contests-cmd:_COMMAND-OPTIONS "wikilink"). В частности,
если указана опция `--session`, то аргумент `SESSION-FILE` — это не
файл, в котором записан сессионный ключ, а сам сессионный ключ.

Информация о задачах выводится в формате CSV по одной задаче на строку.
Информация о каждой задаче имеет следующий вид:

`ID;SHORT-NAME;LONG-NAME`

Здесь `ID` — идентификатор задачи (число \> 0); `SHORT-NAME` — краткое
название задачи; `LONG-NAME` — полное название задачи.
