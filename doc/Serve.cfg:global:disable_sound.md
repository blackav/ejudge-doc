Если эта переменная установлена в 1, то при тестировании программы не
будет выдаваться никаких звуковых сообщений, даже если звуковые
сообщения настроены в программе `run`. Данная переменная предназначена
для использования в ситуациях, когда программа `run` проверяет решения
сразу нескольких турниров (например, основного турнира и интернет
турнира), и для каких-то турниров звуковые сообщения должны быть
разрешены, а для каких-то - нет.

В этом случае можно использовать эту переменную в конфигурационном файле
турниров, для которых звуковые сообщения должны быть отключены.
