Если эта глобальная конфигурационная переменная установлена в
положительное значение, в таблице текущих результатов турнира
отображаются аватары пользователей. Для этого должна быть включена
[поддержка аватаров](поддержка_аватаров "wikilink").

Поддерживается с версии [3.7.0](изменения_в_версии_3.7.0 "wikilink").
