**Описание.** Данный элемент позволяет задавать различные опции для
CGI-программ при работе с данным турниром. Опции в элементе
перечисляются через запятую и могут разделяться произвольным количеством
пробельных символов.

Поддерживаются следующие опции:

|                         |                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `IGNORE_TIME_SKEW`      | Разрешить рассинхронизацию времени между файлом состояния сервера турнира [`serve`](serve "wikilink") и текущим временем при работе турнира. Опция **необходима**, когда турнир управляется программой [`super-serve`](super-serve "wikilink"), поскольку при первом обращении к турниру CGI-программы могут прочитать устревший файл текущего состояния. |
| `DISABLE_TEAM`          | Запретить обращение к турниру с помощью программы [`team`](team "wikilink"). Если опция установлена, при попытке обращения к CGI-программе `team` для данного турнира клиенту будет выдано соответствующее сообщение.                                                                                                                                     |
| `DISABLE_MEMBER_DELETE` | Запретить удаление членов команды и прочих участников при редактировании регистрационных данных пользователя. В этом случае в форме редактирования данных не отображается кнопка `Delete member`.                                                                                                                                                         |

**Пример.**

<client_flags>`IGNORE_TIME_SKEW, DISABLE_MEMBER_DELETE`</client_flags>
