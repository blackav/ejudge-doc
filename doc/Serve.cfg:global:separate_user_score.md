**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, поле *"Calculate and store user-visible
score separately"*.

Если данная конфигурационная переменная установлена в 1, то в базе
данных посылок отдельно хранятся вердикты тестирования для пользователя
и для судей. Пользователю (и в пользовательской таблице результатов)
отображается только пользовательская информация.

Данный режим может быть полезен для турниров, в которых online-проверка
решений участников ведется на подмножестве тестов. Тогда для тестов, не
входящих в online-множество тестов, может быть установлен флаг видимости
"hidden", и информация об этих тестах (и даже об их существовании) не
будет доступна участнику.

Данная конфигурационная переменная доступна, начиная с версии 2.3.18.
