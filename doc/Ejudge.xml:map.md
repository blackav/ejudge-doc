**Описание.** Данный элемент задаёт отображение регистрационных имён
UNIX в регистра- ционные имена системы ejudge. Атрибут `system_user`
задаёт регистрационное имя UNIX. Пользователь с таким регистрационным
именем должен существовать в системе. Атрибут `ejudge_user` задаёт
регистрационное имя системы `ejudge`.

**Пример.**

<map system_user="aaa" ejudge_user="tolik"/>

В данном примере если пользователь `aaa` запустит программу
[`edit-userlist`](edit-userlist "wikilink"), то программа автоматически
получит привилегия доступа к базе пользователей, установленные для
пользователя `tolik` в базе пользователей `ejudge`.
