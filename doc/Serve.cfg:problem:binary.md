**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Problems (serve.cfg)"*, блок *"Abstract problems"*, поле *"Submit is
binary"*, либо страница *"Editing contest"*, вкладка *"Problems
(serve.cfg)"*, блок *"Concrete problems"*, поле *"Submit is binary"*.

Если данная конфигурационная переменная установлена в 1, по данной
задаче принимаются посылки, не являюющиеся [текстовыми
файлами](текстовый_файл "wikilink"). Данная конфигурационная переменная
предназначена для использования в стандартных задачах и в задачах на
написание тестов. Например, если в задаче на написание тестов требуется
сдавать архив тестов, необходимо установить эту переменную, так как
файлы архивов, как правило, не являются текстовыми.

По умолчанию данная конфигурационная переменная не установлена, то есть
принимаются только посылки, являющиеся текстовыми файлами.

Данная переменная наследуюется из абстрактной задачи.
