| **Имя параметра:**         | **`contests_dir`**                                           |
| **Тип содержимого:**       | путь к каталогу                                              |

| **Значение по умолчанию:** | устанавливается программой [configure](configure "wikilink") |

| **Примечание:**            | использование не рекомендуется                               |

**Описание.** Данный элемент задаёт путь к каталогу, в котором хранятся
описания турниров [`contest.xml`](contest.xml "wikilink"). Каждое
описание турнира находится в отдельном файле с именем, равным
шестизначному номеру турнира, например, `000001.xml`.

Рекомендуется устанавливать путь к каталогу описания турниров компиляции
системы `ejudge` с помощью опции `--enable-conf-dir` скрипта
[configure](configure "wikilink"). В этом случае параметр `contests_dir`
в конфигурационном файле использоваться не должен.

**Пример.**

`contests_dir = "/var/ejudge/contests"`
