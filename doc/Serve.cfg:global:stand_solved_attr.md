**Редактирование элемента:** страница *"Editing contest"*, вкладка
*"Global settings (serve.cfg)"*, блок *"Standings table attributes"*,
поле *"HTML attributes for the "HTML attributes for the "Solved"
column"*.

**Описание.** Данная конфигурационная переменная позволяет определить
дополнительные атрибуты столбца «решённые задачи» таблицы результатов
турнира. Значение данной переменной добавляется к элементу `<th>`
заголовка столбцов таблицы и к каждому элементу `<td>`, находящемуся в
столбцах «решённые задачи». Значение данной переменной и элементы `td`
или `th` не разделяются дополнительными пробелами, поэтому значение
данной переменной должно начинаться с пробельного символа.

**Пример.**

`stand_solved_attr = " class=st_solved"`

Такое значение данной конфигурационной переменной приведёт к тому, что
заголовок столбцов «решённые задачи» таблицы будет открываться тегом
`<th class=st_solved>`, а каждая ячейка столбцов задач таблицы — тегом
`<td class=st_solved>`.

**Обратите внимание**, что в текущей версии системы `ejudge`, к
сожалению, невозможно использовать знак «кавычки» (") в значении
конфигурационной переменной.
