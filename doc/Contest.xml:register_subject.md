Данный элемент позволяет задавать тему для писем, высылаемых при
регистрации новых пользователей.

Если не задан ни элемент <register_subject>, ни <register_subject_en>,
тема регистрационных писем задается по умолчанию и зависит от языка
локализации, выбранного пользователем.

Если заданы оба элемента, то элемент <register_subject> используется в
случае, когда при регистрации включен русский язык, а элемент
<register_subject_en> - когда при регистрации включен английский язык.

Если задан один элемент из двух, то используется этот элемент при любом
включенном языке интерфейса.
