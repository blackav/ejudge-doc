Регистрация в ejudge с помощью Telegram-бота позволяет пользователям
создавать аккаунты в ejudge, привязанные к Telegram-аккаунтам
пользователя, и регистрировать эти аккаунты на турниры. Telegram
выполняет роль аккаунта e-mail в традиционной модели создания аккаунтов.

Регистрацию с помощью Telegram-бота можно использовать, если требуется
разрешить свободную регистрацию пользователей, и при этом не настраивать
[аутентификацию OAUTH2](Аутентификация_с_помощью_OAUTH2.md) или
исходящий e-mail на сервере.

Чтобы поддерживалась регистрация с помощью Telegram-бота должны быть
выполнены следующие условия:

- Должен быть [настроен бот
  Telegram](Telegram_bot:_настройка_файла_ejudge.xml.md).
- В ejudge должна быть глобально разрешена возможность создавать
  аккаунты с помощью Telegram-бота с помощью атрибута
  [`enable_telegram_registration`](ejudge.xml:config:enable_telegram_registration.md)
  в файле [`ejudge.xml`](ejudge.xml.md).
- В нужных турнирах должна быть разрешена возможность регистрироваться
  на турнир с помощью Telegram-бота с помощью атрибута
  [`enable_telegram_registration`](contest.xml:enable_telegram_registration.md)
  в конфигурационном файле турнира
  [contest.xml](contest.xml.md).
- Турнир не должен быть [закрыт](contest.xml:closed.md).
- Не должен закончиться [дедлайн
  регистрации](contest.xml:registration_deadline.md).

Чтобы зарегистрироваться в ejudge на турнир пользователь должен открыть
чат с настроенным ботом ejudge и ввести команду `/register`. Появится
меню выбора турниров для регистрации, после чего при необходимости будет
создан новый аккаунт и в Telegram пользователю будут отправлены логин и
пароль от аккаунта и ссылка для дальнейшей регистрации на турнир.

Логины, создаваемые при такой регистрации, имеют вид `u`<i>ID</i>`@tl`,
где <i>ID</i> — это идентификатор пользователя (положительное целое
число) в Telegram.

Поддерживается начиная с версии
[3.12.0](изменения_в_версии_3.12.0.md).
