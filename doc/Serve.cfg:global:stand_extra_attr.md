| **Имя переменной**:        | **`stand_extra_attr`**                  |
| **Содержится в:**          | [`global`](serve.cfg:global "wikilink") |
| **Используется:**          | `serve`                                 |
| **Тип содержимого:**       | *string*                                |
| **Может отсутствовать:**   | *да*                                    |
| **Значение по умолчанию:** | "" (пустая строка)                      |
| **Может повторяться:**     | *нет*                                   |

**Описание.** Данная конфигурационная переменная позволяет определить
атрибуты, с которыми выводится столбец дополнительной информации в
таблице результатов турнира. Столбец дополнительной информации
активируется, если значение конфигурационной переменной
[`stand_extra_format`](serve.cfg:global:stand_extra_format "wikilink")
не пусто. Значение данной переменной добавляется к элементу <tt>

<th>

</tt> заголовка столбцов таблицы и к каждому элементу <tt>

<td>

</tt>, находящемуся в столбцах «дополнительная информация». Значение
данной переменной и элементы `td` или `th` не разделяются
дополнительными пробелами, поэтому значение данной переменной должно
начинаться с пробельного символа.

**Пример.**

`stand_extra_attr = " class=st_extra"`

Такое значение данной конфигурационной переменной приведёт к тому, что
заголовок столбцов «дополнительная информация» таблицы будет открываться
тегом <tt>

<th class=st_extra>

</tt>, а каждая ячейка столбцов задач таблицы — тегом <tt>

<td
class=st_extra>

</tt>.
