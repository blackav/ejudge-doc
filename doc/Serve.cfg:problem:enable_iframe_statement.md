Если у задачи установлен данный конфигурационный параметр, меняется
алгоритм генерации ссылок для ссылок в условии задачи, раскрываемых по
`${getfile}=URL`. Параметры сессии, задачи и другие будут закодированы в
URL, а не в GET-параметры. Это необходимо, чтобы в условии задачи,
импортированном из системы Polygon, и отображаемом в IFRAME, корректно
работали относительные ссылки.

Параметр наследуется из абстрактной задачи.

`[problem]`  
`# ...`  
`enable_iframe_statement`

Поддерживается начиная с версии
[3.10.2](изменения_в_версии_3.10.2 "wikilink").
